字符串str

  • ord(字符串) 返回字符的排序位置
  • chr(整数) 返回排序位置的字符
  • split(" ") 切割成列表
  • str.capitalize() 将字符串的第一个字母变成大写，其他字母变小写
  • str.count(sub) 方法用于统计字符串里某个字符出现的次数
    • sub -- 搜索的子字符串
  • str.endswith(suffix) 方法用于判断字符串是否以指定后缀结尾
  • str.startswith(suffix) 检查字符串是否是以指定子字符串 substr 开头
    • suffix -- 该参数可以是一个字符串或者是一个元素
  • str.find(str) 方法检测字符串中是否包含子字符串 str
    • str -- 指定检索的字符串
    • 如果包含返回开始的索引值，否则返回-1
  • len(string) 返回字符串长度
  • str. swapcase () 将字符串中大写转换为小写，小写转换为大写
  • str.replace(old, new) 把字符串中的 old（旧字符串） 替换成 new(新字符串)

# 子级<   超集>
set03 = {"1","2} 
set03 < set01 # 01中是否存在所有03的数据  true

set03 < set02 # 03中是否存在所有02的数据并数据还存在多余为超集  false

运算法重载

让自定义的类生成对象(实例)能使用运算法: 存在加减乘除

class Person():
    def __init__(self,x):
        self.x = x
      
   	# 计算加法
	def __add__(self, other):
        return other + self.x
    
po2 = Person(3)
print(po2 + 123) # 会执行 po2.__add__(123) 输出 126
