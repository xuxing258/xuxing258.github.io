    9.python运行方式
    	1.交互模式:

输出

迈开学习第一步:输出--第一个python程序--helloword
    输出:说人话就是电脑屏幕 电视机 输出显示的内容
        
在pycharm里面输出字符串
	字符串就是多个字符的集合 由双引号"" 或者单引号 '' 进行包围数据
    字符串里可以包含所有数据
    
print输出字符串的格式如下:
    print("字符串内容")
    print('字符串内容')
    # 在python中单引号/双引号没有区别
    
注意:字符串要放在()中传递给print 让print把字符串显示到屏幕上 这种写法在python当中称为函数
    1.引号和小括号必须都是英文状态下输入 
    2.print需要小写 Python里需注意单词大小写
    3.python国外发明的 所有的符号内容都是英文 (shift键/中英文切换)

print(1000)  # 输出数字
print(1000 + 2000)  # 3000
print("欧阳老师")  # 输出文字
print("+-*/,?")  # 输出符号
print("abcdefg")  # 输出符号
# 整数和字符串不能加号运算
# print("10000"+1000)
