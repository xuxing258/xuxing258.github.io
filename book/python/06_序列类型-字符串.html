数据类型-序列

1.序列介绍

在Python中，序列是一种有序的数据集合。常见的序列类型包括字符串、列表、元组。这些序列类型具有一些共同的特性和操作。

1.1 序列类型都支持共同的操作：

  • 索引访问：使用索引来获取序列中的单个元素，如 sequence[index]。序列[索引]

    # 从左到右序列索引默认从0开始的,正索引值
text1 = "hello world!"
print(text1[0])  # 第一个字符 h
print(text1[11])  # 最后一个字符 ！
print(text1[5])  # 空格，空格也算是字符串里面的一个字符

# 从右往左序列索引默认从-1开始，负索引值
print(text1[-1])  # !
print(text1[-12])  # h

  • 切片操作：使用切片来获取序列中的子序列，如 sequence[start:end:step]。

    # 左闭右开区间
text1 = "hello world!"
print(text1[0:5])  # hello 从0开始的话,0可以不写
print(text1[6:12])  # world!

# 步长
print(text1[0:12:3])  # hlwl 步长不写默认是1

# 反向切片
print(text1[-1:-13:-1])  # !dlrow olleh

  • 长度获取：使用内置函数len()来获取序列的长度，如 len(sequence)。

    text1 = "hello world!"
length = len(text1)
print(length)  # 12表示字符串的长度

print(type(length)) # <class 'int'>


- 成员检查：使用关键字`in`来检查一个元素是否存在于序列中，如
`element in sequence`。

~~~python
text1 = "hello world!"
print("h" in text1)  # True
print("hello" in text1)  # True
print("xiaoming" in text1)  # False

  • 连接操作：使用加号+来连接两个序列，生成一个新的序列，如 sequence1 + sequence2。

    text2 = "hello"
text3 = "world"
text4 = text2 + text3  # 字符串拼接,得到一个新的字符串
print(text4)

  • 重复操作：使用乘号*来重复序列中的元素，生成一个新的序列，如 sequence * n。

    text2 = "hello"
text4 = text2 * 3  # 重复3次之后赋值给到变量text4
print(text4)  # hellohellohello

  
- 迭代：使用`for`循环来遍历序列中的元素，如
  `for element in sequence:`。
  
  ~~~python
  text2 = "hello"
  for i in text2:
      print(i)

  • 元素查找：使用index()方法来查找指定元素在序列中的索引。如果元素不存在，会引发ValueError错误。

    text1 = "hello world!"
print(text1.index("o"))  # 4
print(text1.index("o", 5))  # 第二个o的索引7
print(text1.index("world"))  # 找到world 取首个字符的索引
# print(text1.index("xiaoming")) # 没有找到就报错

  • 计数元素出现次数：使用count()方法来统计指定元素在序列中出现的次数。如 sequence.count("p")

    text1 = "hello world!"
print(text1.count("o"))  # 2 次数,不是索引
print(text1.count("p"))  # 0 没有就是0

  • 最值获取：使用内置函数max()和min()来获取序列中的最大值和最小值，如 max(sequence)和min(sequence)。

    li = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(max(li))  # 9 列表的最大值
print(min(li))  # 1 列表的最小值

  • 排序操作：使用内置函数sorted()对序列进行排序，生成一个新的排序后的序列，如 sorted(sequence)

    li1 = [1000, 20, 31, 2, 11, 55]
# 升序
print(sorted(li1))  # [2, 11, 20, 31, 55, 1000]
# 降序
print(sorted(li1, reverse=True))  # [1000, 55, 31, 20, 11, 2]

2.可变类型与不可变类型

概念：
不可变类型指的是那些一旦创建后，其值就不能被改变的数据类型
可变类型则是指那些创建后仍然可以被改变的数据类型，即可以对其内容进行修改、添加或删除操作而不必创建新对象

不可变类型：数值类型，字符串，元组
可变类型：列表，字典，集合

# 注意:字符串只能通过索引取值,但是无法修改值[字符串在内部存储时不允许对内部元素进行修改,只能创建一个新值]

text1 = "hello world!"
text1[5] = ","  # 不可以  报错

3.1 内置方法

我们在开发程序时需要频繁对数据进行操作，为了提升我们的开发效率， python针对这些常用的操作，为每一种数据类型内置了一系列方法。

语法：字符串.方法名()

  • startswith(): 检查字符串是否以指定的子字符串开头，返回布尔值。

    data1 = "今天是周三"
result = data1.startswith("今天")
print(result)  # True

# 示例
# 根据输入的地址判断户籍
address1 = input("请输入户籍住址:")
if address1.startswith("北京市"):
    print("北京人口")
else:
    print("非北京人口")

  • endswith(): 检查字符串是否以指定的子字符串结尾，返回布尔值。

    data1 = "今天是周三"
result = data1.endswith("周三")
print(result)  # True

# 示例
# 根据输入的文件名判断是否是py文档
file = input("请输入文件:")
if file.endswith(".py"):
    print("python文件")
else:
    print("非python文件")

  • isdigit(): 检查字符串是否只包含数字字符，返回布尔值。

3.2 字符串格式化

字符串格式化是一种在字符串中插入变量或值的方法，以便根据特定的格式要求创建最终的字符串。在Python中，有多种方法可以进行字符串格式化，包括：

  • 使用百分号(%)格式化字符串：这是一种传统的字符串格式化方法，在字符串中使用百分号作为占位符，并使用对应的值进行替换。

    %s：用于字符串
%d：用于整数
%f：用于浮点数，默认保留6位小数
	%.1f:保留一位小数
    %.2f:保留两位小数
        
语法：
"xx%sxx%dxxx%f" % (数据1，数据2，数据3)
注意：%方法需要写在引号内部


name = "欧阳"
age = 18
height = 190.0
# 普通写法
print("我叫" + name + "今年" + str(age) + "岁,身高是" + str(height))
# %方法
print("我叫%s,今年%d岁,身高是%f" % (name, age, height))
print("我叫%s,今年%d岁,身高是%.1f" % (name, age, height))
print("我叫%s,今年%d岁,身高是%.2f" % (name, age, height))
print("我叫%s,今年%d岁,身高是%.2f" % ("欧阳", 18, 180.0))
data = "我叫%s,今年%d岁,身高是%.2f" % ("欧阳", 18, 180.0)
print(data)
print(type(data))

"""
我叫欧阳今年18岁,身高是190.0
我叫欧阳,今年18岁,身高是190.000000
我叫欧阳,今年18岁,身高是190.0
我叫欧阳,今年18岁,身高是190.00
我叫欧阳,今年18岁,身高是180.00
我叫欧阳,今年18岁,身高是180.00
<class 'str'>
"""

  • 使用str.format()方法进行格式化：这是一种更灵活的字符串格式化方法，通过在字符串中使用花括号{}作为占位符，并使用format()方法来进行值的替换。

    语法：
"{}{}{}".format(数据1，数据2，数据3)

name = "欧阳"
age = 18
height = 190.0
# 普通写法
print("我叫" + name + "今年" + str(age) + "岁,身高是" + str(height))
# str.format方法
print("我叫{},今年{},身高{}".format(name, age, height))
print("我叫{},今年{},身高{}".format(1, 2, 3))
data = "我叫{},今年{},身高{}".format(1, 2, 3)
print(data)
print(type(data))

"""
我叫欧阳今年18岁,身高是190.0
我叫欧阳,今年18,身高190.0
我叫1,今年2,身高3
我叫1,今年2,身高3
<class 'str'>
"""

  • 使用f-strings：这是Python 3.6及更高版本引入的一种方便的字符串格式化方法。它允许在字符串前加上字母"f"，并在字符串中使用花括号{}来插入变量或表达式的值。

    语法：直接在字符串前加上F/f前缀，并在花括号{}内放入表达式

name = "欧阳"
age = 18
height = 190.0
# 普通写法
print("我叫" + name + "今年" + str(age) + "岁,身高是" + str(height))
# f-strings方法
print(f"我叫{name},今年{age}岁了,身高{height}")
print(f"我叫{name[0]},今年{age + 10}岁了,身高{height - 10}")
print(f"我叫{'小明'},今年{60}岁了,身高{170.0}")
data = f"我叫{name[0]},今年{age + 10}岁了,身高{height - 10}"
print(data)
print(type(data))

"""
我叫欧阳今年18岁,身高是190.0
我叫欧阳,今年18岁了,身高190.0
我叫欧,今年28岁了,身高180.0
我叫小明,今年60岁了,身高170.0
我叫欧,今年28岁了,身高180.0
<class 'str'>
"""
