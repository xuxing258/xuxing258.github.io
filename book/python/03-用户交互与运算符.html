1. 类型转换

不同的数据类型都有不同的功能，例如：整型可以加减乘除 而 字符串只能加(拼接)和乘法。 如果想要做转换可遵循一个基本规则：想转换什么类型就让他包裹一下。

  • int(x)：将x转换为整数类型。如果x是浮点数，将截断小数部分。如果x是字符串，必须表示一个整数值。

    # str 转 int
# data1 = int("55") + int("100")
# print(data1)  # 155
# data2 = int("张三") # 报错
# data3 = int("1.23") # 报错

# float 转 int
# data4 = int(10.9)  # 向下取整
# print(data4)  # 10

# bool 转 int
# data5 = int(True)
# data6 = int(False)
# print(data5)  # True 转 整数是 1
# print(data6)  # False 转 整数 是 0

  • float(x)：将x转换为浮点数类型。如果x是整数，将转换为相应的浮点数；如果x是字符串，必须表示一个数值。

    # str 转 float
# data1 = float("55") + float("100")
# print(data1)  # 155.0
# data2 = float("张三") # 报错
# data3 = float("1.23") # 不会报错,float有1.23

#  int  转 float
# data4 = float(10)
# print(data4)  # 10.0

# bool 转 float
# data5 = float(True)
# data6 = float(False)
# print(data5)  # True 转 浮点数是 1.0
# print(data6)  # False 转 浮点数 是 0.0

  • bool(x)：将x转换为布尔类型。x可以是任意数据类型，非零数字、非空字符串、非空容器等转换为True，而零、空字符串、空容器等转换为False。

    # int 转 bool
# print(bool(10))  # True
# print(bool(1))  # True
# print(bool(-1))  # True
# print(bool(0))  # False

# float 转 bool
# print(bool(3.14))  # True
# print(bool(-3.14))  # True
# print(bool(0.0))  # False

# str 转 bool
# print(bool("张三"))  # True
# print(bool("666"))  # True
# print(bool(""))  # False
# print(bool("     "))  # True

  • str(x)：将x转换为字符串类型。x可以是任意数据类型。

    # int 转 str
data1 = str(520) + str(1314)
print(data1)  # "5201314"

# float 转 str
data2 = str(520.1) + str(1314.5)
print(data2)  # "520.11314.5"

# bool 转 str
data3 = str(True)
data4 = str(False)
print(data3)  # "True"
print(data4)  # "False"

  • 使用type()函数可以检查变量所指向的数据类型

    a = 10
b = 10.0
c = True
d = "小明"
print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'bool'>
print(type(d))  # <class 'str'>

2. 用户交互

输入和输出是计算机程序与用户交互的基础。输入允许用户提供信息给程序，而输出让程序将其处理结果显示给用户。

2.1 输入

使用input()函数接收用户的输入

input(tipmsg)


data = input("请输入数据:")
print("input接收的内容", data)  # input接收的内容 1000
print(type(data))  # <class 'str'>

tipmsg提示信息，不写就不会有提示信息

input()函数会等待用户输入内容，并将内容作为字符串返回

练习题：

  1. 提示输入姓名和年龄，并将姓名和年龄拼接在一起输出

    name = input("请输入姓名:")
age = input("请输入年龄:")
print(name + age)

  2. 提示输入两个数字，计算两个数的和

    num1 = int(input("请输入第一个数字:"))
num2 = int(input("请输入第二个数字:"))
data = num1 + num2
print(data)

2.2 输出

使用print()函数将括号中的内容输出在控制台

print(10)
print(10.0)
print(True)
print("ABCD")

一次性输出多个值默认会意空格隔开多个值，如果希望改变默认的分隔符，可以通过sep参数进行设置

data1 = "小明"
data2 = "小美"
print(data1, data2)  # 小明 小美
print("Python", "Java", 1000,sep="")  # PythonJava100

默认print在尾部会加上换行符，想要不换行，就可以通过end参数进行设置

print("今天学习的是用户交互", "值", sep="", end=",")
print("今天还学习的是运算符", end=",")

今天学习的是用户交互值,今天还学习的是运算符,

案例：

"""
    汇率转换器
    输入美元，换算成人民币
    7.285
"""
# 1.获取数据： xxx美元
usd = float(input("请输入美元："))  # 字符串——》浮点数
# 2.逻辑计算：xxx人民币 ≈ xxx美元 * 7.285
cny = usd * 7.285
# 3.显示结果：xxx美元 ≈ xxx人民币
print(usd, "美元 ≈", cny, "人民币",sep="")
print(str(usd) + "美元 ≈" + str(cny) + "人民币")

"""
某咖啡店推出以下优惠：
咖啡：每杯15元，买3杯送1杯
任务要求:
输入购买的咖啡数量,计算赠送的咖啡数量,需要支付的费用
"""
# 1.获取数据
coffee = int(input("请输入美式咖啡杯数："))
# 2.逻辑计算
# 赠送的咖啡
free_coffee = coffee // 3
# 需要支付金额的咖啡
coffee_cost = coffee * 15
# 3.显示结果
print("咖啡:", coffee, "杯（送", free_coffee, "杯） 费用：", coffee_cost, "元")


n = int(input())  # 输入实际得到的咖啡
k = 0
while True:
    total = k + (k // 3)  # 总杯数 = 购买杯数 + 赠送杯数
    if total >= n:
        break
    k += 1
cost = k * 15
print("需要购买", k, "杯", "获得", total, "杯,花费", cost, "元", sep="")

运算符说 明用法举例等价形式
=最基本的赋值运算x = yx = y

-=减赋值x -= yx = x - y
*=乘赋值x *= yx = x * y
/=除赋值x /= yx = x / y

3.3 比较运算符

比较运算符说明
>大于，如果>前面的值大于后面的值，则返回True，否则返回False。
<小于，如果<前面的值小于后面的值，则返回True，否则返回False。
==等于，如果==两边的值相等，则返回True，否则返回False。
>=大于等于（等价于数学中的≥），如果>=前面的值大于或者等于后面的值，则返回True，否则返回False。
<=小于等于（等价于数学中的 ≤），如果<=前面的值小于或者等于后面的值，则返回True，否则返回 False。
!=不等于（等价于数学中的 ≠），如果!=两边的值不相等，则返回 True，否则返回 False。
