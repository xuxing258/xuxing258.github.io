函数是编程中用于封装特定功能的代码块，可以重复执行和调用。函数的定义有助于提高代码的可重用性和可维护性，使代码的结构更清晰。

2 函数的定义

在Python中，函数的定义使用关键字def，其语法格式如下:

def 函数名(形参):
    函数体 # 实现特定功能的代码块
    。。。
# 定义了一个发送消息的函数
def send_message():
    print("实现了接收消息的功能,假设写了20行代码来实现!")  # 20行

# 调用函数
send_message()

  • def关键字：def是函数定义的关键字，用于定义一个函数。

  • 函数名：函数名是用来唯一标识函数的名称，遵循变量命名的规则，用于在代码中调用函数。

  • 形参：形参是函数中用于接收外部传入的值的变量，也称为函数的输入。形式参数可以有零个或多个，多个参数之间使用逗号,分隔。

  • 函数体：函数体是实现函数功能的代码块，可以包含一条或多条语句。函数体通常使用缩进来表示代码块的范围。

3 函数的调用

函数的调用是指执行函数体中的代码，实现函数定义中所描述的功能。

# 语法：
函数名(实参)

注意:在python中,函数必须先定义再执行

# 定义了一个函数
def func():
    # 函数体
    print(1)
    print(2)
    print(3)

# 调用函数
func()

print(func)  # <function func at 0x000001D06D9ED280>
print(type(func))  # <class 'function'>

# 补充
# func    是什么 函数对象
# func()  是什么 函数调用
# print   是什么 函数对象
# print() 是什么 函数调用

4 函数参数

# 编写一个模拟发送消息的函数
def send_message(name, message):  # 形参
    # 实现了发送消息的功能,假设20行代码
    info = f"发送了消息给到了{name}同学,内容为:{message}"
    print(info)


send_message("小红", "你作业做的怎么样?")  # 小红
send_message("小白", "你要不要去打游戏")  # 小白


# 求邮箱的长度
result = 0
for i in "123456789@qq.com":
    result += 1
print(result)  # 16

print(len("123456789@qq.com"))  # 16


# 定义求长度的功能函数
def len1(x):
    result = 0
    for i in x:
        result += 1
    print(result)


len1("123456789@qq.com")  # 16

Python函数可以接受不同类型的参数，包括必需参数、关键字参数、默认参数和可变参数。

  • 必需参数（位置参数）：必需参数是在函数定义中按照顺序声明的参数。在调用函数时，必须按照相同的顺序传递参数值。

    # 定义了三个参数的函数 (a,b,c是形式参数-形参)
def func(a, b, c):
    print(a)
    print(b)
    print(c)


# 调用函数并传入值
# ctrl+p 查看函数需要传入的参数
# 实参与形参一一对应
func(1, 2, 3)
func(33, 22, "小明")

  • 关键字参数：关键字参数是在函数调用时使用参数名和值进行传递的参数。这样可以不考虑参数的顺序，只需指定参数名及对应的值。

    def info(name,age,height,address,qq,wechat):
    pass

info()

def func(a, b, c):
    print(a)
    print(b)
    print(c)


# 按照关键字传参
func(a=1, b=2, c=3)
func(b=2, c=3, a=1)
# 混合使用位置参数和关键字参数(位置在前,关键字在后)
func(10, c=30, b=5)

    注意:在调用函数时,实参也可以是按必须或关键字混合使用,但是必须保证关键字参数在必须参数的后面

  • 默认参数：默认参数是在函数定义时为参数提供默认值。如果在函数调用时没有为该参数提供值，则使用默认值。

    def func(a, b, c=30):
    print(a)
    print(b)
    print(c)


# 没有给默认值传值,那就使用默认值
func(1, 2)
# 传递了值,那就是使用传递的值
func(1, 2, 3)
# 默认值参数也可以使用关键字的方式传值(位置在前,关键字在后)
func(1, 2, c=3)

    注意:默认值参数设置的时候在必须参数之后

  • 实参的类型：实参可以传递任意类型

    def func(x):
    print(x)


a = 123  # 传递整数
func(a)  # 123
b = True  # 传递布尔值
func(b)  # True
c = "shijia"  # 传递字符串
func(c)  # shijia
d = [1, 2, 3, 4]  # 传递列表
func(d)  # [1, 2, 3, 4]
# 元组,字典,集合都可以

    # 甚至可以传函数
def num_sum(a, b, c=30):
    print(a)
    print(b)
    print(c)


print(num_sum)  # <function num_sum at 0x0000023DFCB0D280>


def func(x):  # num_sum函数的地址
    print(x)


func(num_sum)  # <function num_sum at 0x0000023DFCB0D280>
