内置高阶函数+装饰器

1.匿名函数

匿名函数，也称为lambda函数，是一种在代码中创建函数的方式，它没有显式的函数定义和命名。匿名函数通常用于一次性的、简单的函数操作。

匿名函数的语法形式是：lambda arguments: expression，其中 arguments 是函数的参数，expression 是函数的返回值。匿名函数可以有多个参数，但只能有一个表达式作为函数体，并且该表达式的结果会被隐式地返回。

# 普通写法求平方的函数
def square(num):
    return num * num

# 匿名函数写法求平方
square = lambda num:num * num


ret = square(3)
print(ret)

2.内置高阶函数

内置高阶函数是指Python内建的一些可以直接使用的、能接受函数作为参数或者将函数作为结果返回的函数。这些函数极大地增强了语言的表达能力和代码的简洁性。

# 扩展内置函数
# sum() # 函数用于计算所有可迭代对象(列表,元组)中的元素的总和
ret1 = sum([1, 2, 3, 4, 5, 6])
print(ret1)  # 21
ret2 = sum([1, 2, 3, 4, 5, 6], 100)
print(ret2)  # 121

# round() # 函数用于将浮点数四舍五入到指定的小数位数
number = 3.7878
ret3 = round(number, 2)
print(ret3)  # 3.79

# abs() # 函数求一个数的绝对值,即不考虑其符号的数值大小
ret4 = abs(-10)
print(ret4)  # 10

# 定义一个函数,求两个数相反数之后再求和
def my_func(num1, num2, f):
    return f(num1) + f(num2)

ret = my_func(1, 2, lambda num:-num)
print(ret)

2.1 map

map()函数：接受两个参数：一个函数和一个可迭代对象。map()函数将传入的函数应用于可迭代对象中的每个元素，并返回一个新的可迭代对象，其中包含应用函数后的结果。

map(function, iterable)
  • function：一个函数，接受一个参数并返回一个值。
  • iterable：一个可迭代对象，例如列表、元组、集合等。
# 将列表中的每个元素转int并得到一个新列表
numbers = ["1", "2", "3", "4", "5"]
# 普通写法
new_num = []
for num in numbers:
    new_num.append(num + "a")
print(new_num)
# 使用map函数
new_num = list(map(lambda x: x + "a", numbers))
print(new_num)

2.2 filter

filter()函数：它接受两个参数：一个函数和一个可迭代对象。filter()函数通过将传入的函数应用于可迭代对象中的每个元素，并返回一个新的可迭代对象，其中包含满足函数条件的元素。

filter(function, iterable)
  • function：一个函数，接受一个参数并返回一个布尔值（True或False）。
  • iterable：一个可迭代对象，例如列表、元组、集合等。
# 筛选出列表中的偶数
numbers = [100, 2, 23, 33, 22, 34, 15]
# 普通写法
# evens = []
# for n in numbers:
#     if n % 2 == 0:
#         evens.append(n)
# print(evens)

# 使用filter方法
evens = list(filter(lambda n: n % 2 == 0, numbers))
print(evens)


# 筛选出字符串列表中长度大于等于6的字符串
names = ["xiaoming", "cheming", "kenan", "nana", "lufei"]
# length = []
# for name in names:
#     if len(name) >= 6:
#         length.append(name)
# print(length)

# 使用filter方法
length = list(filter(lambda n: len(n) >= 6, names))
print(length)

2.3 sorted

sorted()函数：用于对可迭代对象进行排序操作。它接受一个可迭代对象作为参数，并返回一个新的已排序的列表。

sorted(iterable, key=None, reverse=False)
  • iterable：一个可迭代对象，例如列表、元组、集合等。
  • key（可选）：一个函数，用于指定排序的关键字。默认为None，表示使用元素自身进行比较。
  • reverse（可选）：一个布尔值，用于指定排序顺序。默认为False，表示升序排序；设置为True时表示降序排序。
# 对字典列表按照特定字典(age)进行排序
user_info = [
    {"name": "小明", "age": 18},
    {"name": "小白", "age": 30},
    {"name": "车明", "age": 24},
    {"name": "石佳", "age": 20}
]
for a in range(len(user_info)):  # 0 1
    for b in range(0, len(user_info) - 1 - a):  # 0,1,2 30 0,1,2 24
        print(1)
        age1 = user_info[b]["age"]
        age2 = user_info[b + 1]["age"]
        if age1 < age2:
            user_info[b], user_info[b + 1] = user_info[b + 1], user_info[b]
print(user_info)

2.4 max和min

max()和min()函数：用于获取可迭代对象中的最大值和最小值。

max(iterable, key=None)
min(iterable, key=None)

user_info = [
    {"name": "小明", "age": 18},
    {"name": "小白", "age": 30},
    {"name": "车明", "age": 24},
    {"name": "石佳", "age": 20}
]

max_age = max(user_info,key=lambda n:n["age"])
print(max_age) # {'name': '小白', 'age': 30}
min_age = min(user_info,key=lambda n:n["age"])
print(min_age) # {'name': '小明', 'age': 18}
  • iterable：一个可迭代对象。
  • key（可选）：一个函数，用于指定比较的关键字。默认为None，表示使用元素自身进行比较。
