1. 注释

解释器会忽略注释内容

  • 单行注释

print("小帅同学")  # 注释内容
# 还能继续写吗

快捷键:选中你想注释的代码行,然后按住ctrl+?

  • 多行注释

    """

2. 初始数据类型

数据类型是编程语言中用来定义不同种类数据的分类，不同的数据类型具有不同的特征和操作。

2.1 整数类型 int

整型，整数，用来表示整数相关的数据（年龄，手机号，存款等）

print(2025)  # 2025

print(10 + 10)  # 20

print(1 * 10) # 10

print(10 / 2)  # 5.0

print(10 % 3)  # 1

print(2 ** 3)  # 8


# python3.6及更高的版本支持数字分割符
print(1000000000)
print(10_0000_0000)

2.2 浮点数类型 float

表示带有小数部分的数字，用来存储人的身高，体重，薪资等小数相关的数据。

  • 十进制表示：就是我们平时看到的小数形式，例如：3.14 31.4 314.0 0.314

    print(3.14 + 3.14)  # 6.28

  • 科学计数法：当浮点数非常大或者非常小时，可以使用科学计数法，科学技术法使用e或E后跟指数来表示10的幂

    # 及时看上去是整数,但是使用科学计数法的方式,那么在程序中就是一个浮点数
print(3.14e2)  # 3.14乘以10的2次方  结果为314.0
print(3.14e-3)  # 3.14乘以10的-3次方  结果为0.00314

2.3 布尔类型 bool

布尔类型中有两个值：True/False

用来表示真(True)或假（False）

print(1 > 2)  # False
print(1 < 2)  # True

案例:

if 25 >= 18:
    print("成年人")
else:
    print("未成年人")

2.4 字符串类型 str

字符串在Python中是一种用于存储文本信息的数据类型，可以包含字母，标点，特殊符号以及全球各种文字

  • 单行字符串

    print("你好,你在做什么")
print('你好,同学')
print('你好,"同学')
print("你好,'同学")
# 输出结果
你好,你在做什么
你好,同学
你好,"同学
你好,'同学

  • 多行字符串

    print("""你好哈
同学
你准备去干什么呀
""")
# 输出结果
你好哈
同学
你准备去干什么呀


print("""文本
内容""")
print('''文本
内容''')

  • 转义字符：转义字符是一种特殊的字符序列，用于表示在字符串中具有特殊意义的字符。在python中，转移字符使用反斜杠（\）作为前缀，后跟特定的字符，以表示特殊含义。

    """
1.\':表示单引号（'）
2.\":表示双引号（"）
3.\\:表示反斜杠（\）
4.\n:表示换行符
5.\t:表示制表符
"""
print("文本内容'\"中")
print('文本内容\'"中')
print("你好,同学\n你作业写了吗")
print("你好,同学\t你作业写了吗")
# 输出结果
文本内容'"中
文本内容'"中
你好,同学
你作业写了吗
你好,同学	你作业写了吗

  • 原始字符串：在字符串的开头加上r前缀，就变成了原始字符串\不会被当成转移字符，所有的内容都保持”原汁原味“的样子。

    print("C:\86199\Desktop\\n.py")
print(r"C:\86199\Desktop\n\n\n\n\n\n\n\n\n.py")

  • 字符串的基本使用

    • 加，两个字符串可以通过加号拼接起来

      print("abcd" + "qwer") # abcdqwer
# print("abcd" + 100) # 不可行的,类型不一样
print("abcd" + "100") # abcd100

    • 乘，字符串和整型进行相乘，那么字符串重复出现n次并拼接起来

      print("学习python使我快乐\n" * 3)
print("学习python使我快乐\n" + "学习python使我快乐\n" + "学习python使我快乐\n")

# 输出结果
学习python使我快乐

学习python使我快乐

学习python使我快乐

3. 变量

变量，给数据在程序中起别名，以后可以通过变量名来操作其对应的值（数据）

计算机用来存储数据的盒子，想用这个数据，就直接拿这个盒子就好了

语法：
变量名 = 值（数据）

# 将文本数据赋值给到变量text,从此以后,这个text就表示300个字
text = "300个文字的小说"
print(text)

x = 10
print(x)  # 10
x = x + 1  # 自增
print(x)  # 11

# 多重赋值
name, age, height = "小明", 18, 180.0
print(name)  # 小明
print(age)  # 18
print(height)  # 180.0


# 链式赋值 : 把同一个值同时赋值给到多个变量
a = b = c = 10
print(a)
print(b)
print(c)


# 交叉赋值: 将变量的值进行交换
a = 10
b = 5
a, b = b, a
print(a)  # 5
print(b)  # 10


# 表达式
number = 5 + 10 + 10 + 10
print(number + 1)  # 36
flag = 1 > 2
print(flag)  # False
# address地址
address = "湖南" + "岳阳" + "岳阳楼"
print(address + "abcd") # 湖南岳阳岳阳楼abcd

3.1 命名规则

规则是指明确的，强制性的原则或要求，必须遵守

  • 变量名只能由字母，下划线，数字组成，且不能以数字开头

    错误示范
1name = "小明"
na me = "小明"
name? = "小明"

  • 变量的名字是严格区分大小写的

    # 这是两个不同的变量
age = 18
AGE = 20
print(age)
print(AGE)

  • 不能使用Python内置的关键字

    ['False', 'None', 'True', 'and', 'as', 'assert', 'async', 'await', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']

    # 错误示范
def = 100
as = 1000

3.2 命名规范

规范是一种约定旨在提供一致性和最佳实践的建议，但不强制要求遵守。

  • 见名知意

    a = "小明"
b = 18
name = "小明"
age = 18

  • 下划线连接命名（小写字母）

    user_name = "小帅"
user_age = 18
user_money = 1000000

  • 常量（不会改变的量），可以全部使用大写字母，下划线连接命名

练习题:

name1 = "小明"
name*1 = 10
1_age = 18
age_1 = 19
