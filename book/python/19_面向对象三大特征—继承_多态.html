1.上节课回顾

假设我们有一个表示矩形(rectangle)的类，其中长度(length)和宽度(width)需要验证为正数，并且我们可以通过这个长度和宽度来计算矩形的面积。

class Rectangle:

    def __init__(self, length, width):
        # 不是直接给对象添加实例属性得位置
        self.length = length
        self.width = width
        # if length > 0 and width > 0:
        #     self.__length = length
        #     self.__width = width
        # else:
        #     raise ValueError("长度和宽度必须为正数")  # 抛出异常

    @property
    def length(self):
        return self.__length

    @length.setter
    def length(self, value):
        if value > 0:
            self.__length = value # 添加对象属性的位置了
        else:
            raise ValueError("长度必须为正数")  # 抛出异常

    @property
    def width(self):
        return self.__width

    @width.setter
    def width(self, value):
        if value > 0:
            self.__width = value
        else:
            raise ValueError("宽度必须为正数")

    @property
    def area(self):
        return self.__length * self.__width


# 创建Rectangle类的对象
# rect = Rectangle(10, 5)
rect = Rectangle(-10, 5)
# 访问实例属性
print(rect.length)
print(rect.width)
print(rect.area)
# 修改实例属性的时候，必须得传正数
# rect.length = -100
# rect.width = -100

2. 继承

继承：生活中，儿子继承父亲的财产

面向对象编程的一个重要特性是继承。继承允许你创建一个类（称为子类或派生类），它继承另一个类（称为父类或基类）的属性和方法。这样可以重用代码并减少重复，同时还能根据需要添加新的特性和方法。

在python3中所有类都继承了一个object类,object被称为"顶级基类"或"根类"

2.1 单继承

一个子类继承一个父类

父类隐藏属性子类是无法直接继承的

class Animal(object):
    """动物类"""
    def sound(self):
        print("发出叫声")

    def eat(self):
        print("吃东西")

class Dog(Animal):
    """狗类"""
    def f(self):
        print("我是f方法")

class Cat(Animal):
    """猫类"""
    pass
# 子类可以使用父类的方法
c = Cat()
c.sound()
c.eat()

d = Dog()
d.sound()
d.eat()
# 父类无法使用到子类的方法
# a = Animal()
# a.f()

# isinstance(对象,类) 检查对象是否由指定的类实例的
print(isinstance(d, Dog))  # True
print(isinstance(d, Cat))  # False
print(isinstance(d, Animal))  # True

# issubclass(子类,父类) # 检查是否有继承关系
print(issubclass(Dog, Animal)) # True
print(issubclass(Dog, object)) # True
print(issubclass(Dog, Cat)) # False

2.2 重写父类中的属性与方法

  • 子类覆盖父类中的方法

    class Animal(object):
    """动物类"""

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sound(self):
        print("发出叫声")

    def eat(self):
        print("吃东西")


class Dog(Animal):
    """狗类"""

    def __init__(self, color):
        self.color = color

    def sound(self):
        print("汪汪叫")


# 优先使用子类的方法,子类没有,就找父类
d = Dog("黄色")
d.sound()
d.eat()

  • 子类继承父类方法并重写

    super()函数用于调用父类的方法,在子类中,通过super().父类方法可以调用父类的方法

    class Animal(object):
    """动物类"""

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sound(self):
        print("发出叫声")

    def eat(self):
        print("吃东西")


class Dog(Animal):
    """狗类"""

    def __init__(self, name, age, color):  # "小白",3,"白色"
        super().__init__(name, age)
        self.color = color

    def sound(self):
        # super(Dog, self).sound()
        super().sound()  # "发出叫声"
        print("汪汪叫")


d = Dog("小白", 3, "白色")
print(d.name)
print(d.age)
print(d.color)
d.sound()

2.3 多层继承

多层继承:子类继承父类,父类继承对应的父类(爷爷类)......

class A:
    def __init__(self):
        self.money = 5000

    def car(self):
        print("奔驰")


class B(A):
    def __init__(self):
        self.money = 1000

    def car(self):
        super().car()
        print("保时捷")


class C(B):

    def car(self):
        super().car()
        print("电车")


c = C()
print(c.money)
c.car()

2.4 多继承

一个子类继承了多个父类

在多继承中同一个方法或者属性遵循就近继承(在父类中从左到右)


class Horse:
    """马类"""

    def body(self):
        print("体型健硕")
    def run(self):
        print("奔跑速度快")

class Donkey:
    """驴类"""
    def body(self):
        print("体型较小")

    def f(self):
        print("我是f方法")

class Mule(Horse,Donkey):
    """骡子类"""
    pass

# 通过Mule创建一个对象
mule = Mule()
mule.body()
mule.run()
mule.f()

# 查看继承顺序 mro
print(Mule.__mro__)
# (<class '__main__.Mule'>, <class '__main__.Horse'>, <class '__main__.Donkey'>, <class 'object'>)
print(Horse.__mro__)
# 查看继承的父类,返回一个元组
print(Mule.__bases__)
print(Horse.__bases__)

# 返回一个父类
print(Mule.__base__) # <class '__main__.Horse'>

  • 鸭子类型

    鸭子类型是动态类型的一种风格,在鸭子类型中,关注点在与对象的行为,能做什么;而不是关心对象所属的类型

    class Animal(object):
    """动物类"""

    def sound(self):
        print("发出叫声")


class Dog(Animal):
    """狗类"""

    def __len__(self):
        return 100

    def sound(self):
        print("汪汪叫")


class Cat(Animal):
    """猫类"""

    def sound(self):
        print("喵喵叫")


dog = Dog()
cat = Cat()


def make_sound(animal):
    animal.sound()


make_sound(dog)
make_sound(cat)

print(len("12345"))
print(len([1, 2, 3, 4, 5]))
print("1234".__len__())
print(len(dog))

面向对象的三大特征:

  • 封装:
    • 封装是指将数据(属性)和操作数据的方法(函数)组合在一起,形成一个类,并对外部隐藏类的内部实现细节,这样可以提高的代码安全性和可维护性
  • 继承:
    • 允许一个类从另一个类继承属性和方法,从而实现代码的复用和拓展
  • 多态:
    • 允许使用统一的接口来操作不同的对象,从而使得代码更加的灵活和可扩展
