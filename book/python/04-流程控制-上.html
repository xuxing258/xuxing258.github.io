运算符说明

在计算机中，硬盘、内存和CPU是三个基本的组件，它们各自承担着不同的功能：

  • 硬盘：硬盘是用来持久存储数据的。当你关闭计算机时，保存在硬盘上的数据依然存在，比如你的文件、程序等。
  • 内存：内存是用来暂时存储数据的。内存的访问速度非常快，但它是易失性的，即当计算机关闭或重启时，存储在内存中的数据会丢失。
  • CPU：负责执行程序指令，处理数据。CPU直接从内存中读取数据进行处理，因为内存的响应速度足以匹配CPU的处理速度。

在程序运行时，变量是存储在内存中的。当你在程序中定义一个变量时，会在内存中分配空间来存储这个变量的值。这允许CPU能快速访问这些变量。变量只是值在内存中位置的一个引用。当你声明x = 10，x不直接存储值10，而是指向存储值10的内存位置。

可以通过id()函数来访问

# number1 = 10
# number2 = 10
# print(id(number1)) # 显示number1引用的对象的内存地址标识
# print(id(number2))

num2 = 10
num3 = 10.0
print(num2 == num3)  # True
print(num2 is num3)  # False
print(num2 is not num3)  # True
print(id(num2))
print(id(num3))

双等号比较的是value是否相等，而is比较的是id是否相等

  • id相同，内存地址必定相同，这意味着这两个对象实际上是同一个对象，因此type和value必定相同
  • value相同type不一定相同，但id可能不同

在Python中解释器预先创建了一定范围内的整数对象，并将它们放在一个特殊的内存区域，这就是所谓的“整数池”，也常被称为“小整数缓存”。

Python中的整数池包括了从-5到256的所有整数。这意味着当你创建这个范围内的任何整数时，Python不会每次都创建一个新的对象，而是直接从整数池中返回已经存在的对象的引用。

1.6 逻辑运算符

逻辑运算符含义基本格式说明
and逻辑与运算，等价于数学中的“且”a and b当 a 和 b 两个表达式都为真时，a and b 的结果才为真，否则为假。
or逻辑或运算，等价于数学中的“或”a or b当 a 和 b 两个表达式都为假时，a or b 的结果才是假，否则为真。
not逻辑非运算，等价于数学中的“非”not a如果 a 为真，那么 not a 的结果为假；如果 a 为假，那么 not a 的结果为真。相当于对 a 取反。

在Python中，逻辑运算符支持短路运算。短路运算是一种特殊的逻辑运算，当结果已经可以确定时，就不再执行后续的表达式，从而提高程序的效率。

短路与运算（and）：

  • 如果第一个表达式为False，则整个表达式的结果为False，不再计算后续表达式。
  • 如果第一个表达式为True，则继续计算后续表达式，最终结果取决于最后一个表达式。

短路或运算（or）：

  • 如果第一个表达式为True，则整个表达式的结果为True，不再计算后续表达式。
  • 如果第一个表达式为False，则继续计算后续表达式，最终结果取决于最后一个表达式。
score1 = 90 > 60  # True # 语文成绩
score2 = 55 > 60  # False # 数学成绩
print(score1 and score2)  # False 两者都满足给你奖励
print(score1 or score2)  # True 只有有一个满足给你奖励
print(not score1)  # False
print(not score2)  # True

算术运算符优先级大于比较运算符

程序有3种结构：顺序结构、分支结构和循环结构。若是在程序中没有给出特别的执行目标，则默认自上而下一行一行地执行该程序，这类程序的结构就称为顺序结构。之前我们写的代码都是顺序结构。

2.分支结构

分支结构是流程控制中的一种形式，用于根据条件的真假选择不同的执行路径。在分支结构中，程序根据条件的结果执行不同的代码块。可分为单分支、双分支、多分支。

2.1 单分支结构

单分支结构是最简单的分支结构形式，它只有一个条件，根据条件的结果选择是否执行特定的代码块。当条件为真时，执行相应的代码块；当条件为假时，跳过代码块继续执行后续代码。

if 条件:
    条件为真时执行的代码块

name = input("请输入你的姓名:")
if name == "欧阳老师":
    # 一个tab键
    print("原来是世界第一帅来了")
    
    
age = int(input('请输入你的年龄：'))
if age >= 18: # 冒号是必须要写的
    print('成年')
  • 代码块由具由相同缩进量的若干条语句组成。
    • 缩进通常使用四个空格或一个制表符进行。
    • 在同一个代码块中，所有的语句必须具有相同的缩进级别。
  • if语句的最后要有有冒号:，不要忘记。

2.2 双分支结构

双分支结构有两个分支路径，根据条件的真假选择不同的执行路径。当条件为真时，执行一个代码块；当条件为假时，执行另一个代码块。

if 条件:
    条件为真时执行的代码块
else:
    条件为假时执行的代码块

age = int(input('请输入你的年龄：'))
if age >= 18:  # 如果不成立 就是小于18
    print('成年')
else: # else不需要你写表达式了
    print("未成年")
    
    
user_name = input("用户名:")
password = input("密码:")
if user_name == "xm" and password == "123":
    print("登录成功")
else:
    print("登录失败")

2.3 多分支结构

多分支结构有多个分支路径，根据不同的条件结果选择不同的执行路径。条件逐个进行判断，一旦有条件为真，执行相应的代码块并跳过剩余的条件判断。如果所有条件都为假，则执行else代码块（可选）。

if 条件1:
    条件1为真时执行的代码块
elif 条件2:
    条件2为真时执行的代码块
elif 条件3:
    条件3为真时执行的代码块
......
else:
    所有条件都为假时执行的代码块


score = int(input("请输入成绩:"))
if score >= 90:
    print("优秀")
elif score >= 80:
    print("还不错")
elif score >= 70:
    print("一般般")
elif score >= 60:
    print("这也不太行呀")
else:  # 如果上面的都不成立,那么就直接执行else
    print("免不了一顿打")
  • elif可以有0个或多个。
  • else可以有0个或1个，且只能放在if语句的最后。
  • elif和else不能单独使用

条件可以是一个单一的值或者变量，也可以是由运算符组成的复杂语句。不管条件的结果是什么类型。if都能判断它是否成立（真或者假）
