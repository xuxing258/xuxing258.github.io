video视频属性

  • src 视频路径
  • width 视频的宽
  • height 视频的高
  • controls 显示控件 /kənˈtrəʊlz/
  • muted 视频禁音 /ˈmjuːtɪd/
  • autoplay 视频自动播放
  • psoter="路径" 视频播放前广告 搜特
  • loop 循环播放
  <video src="./vedio/1.mp4" width="200"  height="200" controls  ></video>

// 高版本浏览器想要视频自动播放 设置 autoplay 并静音 muted
// 或者 服务器坏境下 设置autoplay 能自动进行播放  
// 服务器环境下能进行缓存

 video.src       设置或返回视频的 src 属性的值。
        .currentTime  设置或返回视频中当前播放位置
        .muted        设置或返回是否关闭声音。
        .playbackRate 设置或返回视频播放的速度。
        .height       设置或返回视频的 height 属性的值。
        .width        设置或返回视频的 width 属性的值。
        .loop         设置或返回视频是否应在结束时再次播放。
        .paused       返回视频是否暂停播放。   /pɔːzd/
        .poster       设置或返回视频的 poster广告 属性的值
			video.poster = "图片地址"
        .volume       设置或返回视频的音量 => 默认为1最大音量

// 不可设置
.ended        返回音频的播放是否已结束。
.duration     返回音频的长度（以秒计）。

视频事件

onvolumechange  每当音量改变时（包括将音量设置为静音）时运行的脚本。
onplay          当媒介已就绪可以开始播放时运行的脚本。
onpause         当媒介被用户或程序暂停时运行的脚本。
ontimeupdate    当播放位置改变时（比如当用户快进到媒介中一个不同的位置时）运行的脚本。
onratechange    播放速度变化后触发
onended         当视频播放完成时
onloadeddata    首帧已经加载完成触发
oncanplay       当文件就绪, 可开始播放时运行的脚本（缓冲已足够开始时）。

object-fit 语法

设置视频或图片再元素中填充区域高宽设置

  • contain: 默认值 在此容器内保持比例缩放以适应容器, 此参数可能会在容器内留下空白
  • fill: 内容拉伸填满整个元素, 不保证保持原有的比例
  • cover: 内容拉伸填满整个元素,按比例放大如果对象的宽高比与内容框不相匹配，该对象将被剪裁以适应内容框
  • none: 保持原有尺寸比例

  • loop 循环播放

音频播放新政策

  1. 音频被静音或其音量设置为0
  2. 用户和网页已有交互行为（包括点击、触摸、按下某个键等等）
  3. 页面允许使用自动播放功能
  <audio src="音频文件" autoplay controls  ></audio>

// 服务器坏境下 设置autoplay 能自动进行播放   能进行缓存
// 必须用户交互

source元素(设置备用视频)

鉴于浏览器对图像文件格式和媒体文件格式的支持不同，该元素通常用于以多种文件格式提供相同的媒体内容，以便与多种浏览器兼容

或用于多个视频时, 选择能播放成功的视频 (设置备用视频)

<!-- 选择浏览器支持的格式进行播放或 能播放的视频进行播放 -->
<video  width="300px" height="200px" controls muted   autoplay loop >
    <!-- 处理浏览器视频兼容问题 -->
    <source src="./supernova.mp4" type="video/mp4" />
    <source src="./supernova.mp4" type="video/webm" />
    
    <!-- 假如上方视频引入不成功,无法播放;  下方视频能引入成功则播放下方视频 -->
    <source src="./aespa.mp4" type="video/mp4" />
</video>

字幕

字幕使用格式是 vtt => webvtt字幕

<track default kind="captions" srclang="zh-cn" src="./11.vtt" />

使用 track 标签读取 vtt格式文字

 <video id="video" src="./vedio/1.mp4" width="600" autoplay muted loop  controls>
    <track default
    kind="captions"
    srclang="zh-cn"
    src="./11.vtt" />
</video>

track属性

  1. default 该属性指示应该启用该轨道，除非用户的首选项指示另一个轨道更合适
  2. kind captions隐藏式字幕提供了音频的转录和翻译
  3. src 文件路径
  4. srclang 轨道文本数据的语言 zh-cn || en 等
