linux

文件

  • 切换目录 cd 目录

  • 查看文件

    • ls [目录]
    • ll: 查看文件详细信息

  • 创建文件及目录

    • touch 文件名

      mkdir 目录(文件夹)

  • 删除文件及目录

    • rm 文件:删除文件

      rm -rf 目录:删除目录及里面的内容

  • 复制文件及目录

    • cp 源文件 目录路径/目标文件名:复制文件

      cp -r 原目录 目录目录:复制目录

  • 移动文件及目录

    • mv 源文件 目录路径/目标文件名:移动文件

      mv 原目录 目标目录:移动目录

      mv 源文件 目标文件名(在同一级目录下操作):重命名

  • 重定向:把原本输出在屏幕上的内容,输出到文件中

    • 覆盖模式:>

      追加模式:>>

  • 管道命令:|

    • | 将前一个命令的结果作为后一个命令的参数使用

  • 查看文件内容

    • cat 文件名

      more 文件名:分屏查看文件内容

文件打包和压缩命令:tar

选项说明
-c创建新的打包文件
-v显示打包或者解包的文件的详情
-f使用打包文件或者设备,这个选项是必选
-z用gzip格式来压缩或者解压文件(.gz)
-j用bzip2格式来压缩或解压文件(.bz2)
-x解压
-C解压缩到指定目录

格式的压缩和解压

  • gz格式的压缩
    • tar -zcvf 压缩后的名字.tar.gz 要压缩的文件和目录
  • gz解压
    • tar -zxvf 压缩包的名字.tar.gz -C 要解压的目录
  • bz2格式的压缩
    • tar -jcvf 压缩后的名字.tar.bz2 要压缩的文件和目录
  • bz2格式的解压
    • tar -jxvf 压缩包的名字.tar.bz2 -C 要解压的目录

chmod命令:修改文件权限

  • 文件权限

  • 九位字母

    • r:读, w:写, x:执行, -:无权限

      1-3:创建文件的用户的读写执行

      4-6:同组用户的读写执行

      7-9:其他用户的读写执行

      例:rw-rw-r--

      创建文件的用户读写, 同组用户有读写, 其他用户只有读权限

    • 九位二进制

      • 1:有权限, 0:无权限

        例:110110100

        创建文件的用户读写, 同组用户有读写, 其他用户只有读权限

    • 三位八进制

      • 4:读, 2:写, 1:执行

        第一位:创建文件的用户

        第二位:同组用户

        第三位:其他用户

        例: 754

        创建文件的用户读写执行, 同组用户有读和执行, 其他用户有读权限

  • 字母法修改

  • 角色说明
    u创建文件的用户
    g同组用户
    o其他用户
    a所有用户
    权限操作说明
    +增加权限
    -撤销权限
    =设置权限

chmod u=rw, g+w, o=r a.txt

数字法修改

例:chmod 754 a.txt

软件包管理:apt命令

apt命令提供了查找,安装, 升级,删除某一个软件甚至全部软件包的命令

apt [选项] [命令] [包名]

  • 列出所有可更新的软件清单:sudo apt update
  • 升级软件包: sudo apt upgrade
  • 列出可更新的软件包及版本信息:apt list --upgradeable
  • 安装指定软件:sudo apt install 包名
  • 安装多个软件包:sudo apt install 包名1 包名2
  • 删除软件:sudo apt remove 包名

安装python3版本环境:sudo apt install python3

安装pip:sudo apt install python3-pip

pip3 install 库名

vim编辑器

  • 三种工作模式

    • 命令模式

      启动vim编辑器,默认进去的就是命令模式.在该 模式下,键盘上输入的字符会被当做命令

    • 编辑命令

      • 在命令模式下输入i,a,o进入编辑模式

        在该模式下,只有用于录入文本

        按esc键回退到命令模式

    • 末行模式

      • 在命令模式下按shift加;(也就是冒号)进入到末行模式下

        在该模式下,对文件进行管理

        按esc退回到命令模式

在命令模式下的操作

  • 移动光标
    • 上下左右移动
    • ^移动至行首,$移动至行尾
    • G移动至文件最后一行
    • gg移动至文件的首行
  • 删除命令
    • x:删除光标所在字符
    • dd:删除光标所在一行
  • 复制和粘贴
    • yy:复制光标所在一行
    • nyy:复制包括光标所在的n行
    • p:粘贴一次
    • np:粘贴n次

末行模式下的vim操作

  • 文件保存和退出
    • w:保存
    • q:退出(文件被改变不能直接q退出)
    • wq:保存并退出
    • q!:不保存退出
  • 显示和隐藏行号
    • set nu:显示
    • set nonu:隐藏

文件操作流程

  • 默认输入vim 文件名进入编辑器
  • 按i进入到编辑模式
  • 编辑完文件之后按esc键退回到命令模式
  • 输入shift加;进入末行模式
  • 输入wq保存并退出

远程登录

非本地,不同主机之间通过网络实现登录

  • ssh实现远程登录

    • linux系统:
    • 安装服务:sudo apt install openssh-server
    • 安装客户端:sudo apt install ssh或者sudo apt install openssh-clients

  • windows系统安装

    安装软件WinSCP(上传到百度网盘)

  • 客户端远程登录服务端

    • 启动服务器 systemctl start sshd.service
    • 看服务器运行情况 systemctl status sshd.service

  • windows系统远程登录linux系统

    • 查看linux系统的ip地址:ifconfig (需要通过sudo apt install net-tools)

直接选择windows系统的文件拖动到linux系统实现上传文件

web服务器

  • Apache:是开源老牌的web服务器,在世界上排名第一,几乎可以运行在所有广泛使用的计算机平台上.
  • Nginx:是一款用于高并发服务的服务器,只要用于高并发网站,静态服务器,下载服务器等.性能更高,支持负载均衡,反向代理等
  • Tengine:是淘宝开发团队基于nginx开发的一款web服务器
