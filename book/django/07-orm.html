作业讲解

1. 编写一个视图函数，接收一个名为 name 的参数（通过 URL 传递），并在页面上显示 “你好，[name]！欢迎访问本网站。” 的个性化问候语。例如，当访问 /user/John/ 时，页面显示 “你好，John！欢迎访问本网站。”

# urls.py
re_path('users/(?P<name>\w+)', views.index),


# views.py
def index(request, name):
    return HttpResponse('你好，%s！欢迎访问本网站。' % name)

2.创建一个包含一个文本输入框和一个提交按钮的 HTML 表单。当用户提交表单时，在视图中接收输入的文本，并将其转换为大写后显示在新的页面上。例如，用户输入 “hello”，提交后页面显示 “HELLO”。

# urls.py
path('test/', views.test),

# views.py
def test(request):
    if request.method == 'POST':
        text = request.POST.get('text')
        return HttpResponse(text.upper())
    return render(request, 'test.html')

# text.html
<form action="" method="post">
    <p>请输入文本:<input type="text" name="text"></p>
    <p><input type="submit"></p>
</form>

4.创建一个允许用户上传文本文件的表单。在视图中接收上传的文件，读取文件内容，并保存
# urls.py
path('file/', views.file),

# views.py
def file(request):
    if request.method == 'POST':
        file_obj = request.FILES.get('file')  # get函数没有获取到字典中的值返回None
        if file_obj:  # 文件是否获取到
            with open(file_obj.name, 'wb+') as f:
                for i in file_obj:
                    f.write(i)
            return HttpResponse('okok')
        return HttpResponse('没有选择文件')
    return render(request, 'file.html')

# file.html
<form action="" method="post" enctype="multipart/form-data">
    <input type="file" name="file">
    <input type="submit">
</form>

静态文件

静态文件:前端已经写好了的,能够直接调用使用的文件(写好的js文件,css文件或者用到的图片,第三方的前端框架等等)

# django默认不会帮你创建静态文件夹,需要我们自己手动创建.一般情况下,会在根目录下创建一个叫做static文件夹,用来存放静态文件
# 前端能够通过url看到对应的资源,是因为后端提前开设了该资源的接口.如果访问不到该资源,是因为后面没有开设接口
# 想要在前端页面看到静态资源
# 1在根目录下创建static文件夹,在static文件夹中存放对应的静态文件
	-static文件夹
    	--css文件夹
        --js文件夹
        --images文件夹
        --前端框架文件夹
        
# 2.静态文件配置:在settings.py中配置static文件夹的地址
STATICFILES_DIRS = [
    (BASE_DIR / 'static')
]

# 3.在html页面中引用静态文件
<!--使用外部的css文件和js文件-->
<link rel="stylesheet" href="/static/bootstrap-3.4.1-dist/css/bootstrap.min.css">
<script src="/static/js/jquery-3.7.1.min.js"></script>
<!--导入jquery之后才能使用bootstrap中的js效果-->
<script src="/static/bootstrap-3.4.1-dist/js/bootstrap.min.js"></script>
<!--引用图片-->
<img src="/static/images/yue.png" alt="" width="200px">

基本使用

  • 连接数据库
# settings.py中
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'day7',  # 连接一个存在的数据库
        'USER': 'root',  # root用户名
        'PASSWORD'; '123456',  # root账号的密码
        'HOST': '127.0.0.1', # 主机默认是'127.0.0.1'或者'localhost'
        'PORT': 3306,  # 端口号.默认是整型3306  
        'CHARSET': 'utf8'  # 数据库编码类型,取决于创建数据库时数据库的默认编码
    }
}
  • 创建模型类
# 在models.py中创建模型类
from django.db import models


# Create your models here.
class Student(models.Model):
    # id int auto_increment primary key
    id = models.AutoField(primary_key=True)
    # name varchar(32)
    name = models.CharField(max_length=32)
    # age int
    age = models.IntegerField()
  • 进行数据库迁移
# 只要对models.py中进行的操作就需要进行数据库迁移两条命令
python manage.py makemigrations:将操作保存到migrations文件夹中
python manage.py migrate:将迁移记录同步到数据库中
