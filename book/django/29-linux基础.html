inux基础

linux终端介绍

点击linux系统左下角的九个点能够打开所以的应用,找到终端,把它固定到收藏栏中

  • 命令提示符

    python@ubuntu:~$

    n python:用户名

    n ubuntu:主机名称

    n ~:表示当前用户所在文件夹

    n $:表示当前用户类型,#表示管理员,$表示普通用户

  • 终端控制字体大小

    n 临时设置字体大小

    ctrl加-:缩小字体

    ctrl加shift加+:增大字体

    • 永久设置字体大小
      • 在终端命令界面点击编辑-->首选项-->文本

linux命令使用

  • pwd:查看当前目录

  • who:显示当前登录用户

  • sudo:以管理员身份运行命令

  • tree:查看目录结构

  • ifconfig:查看网卡配置信息

  • ctrl加l:清空命令终端内容

linux命令格式

命令 [-选项] [参数]

命令:比如pwd, who

选项:可以有零个,一个或者多个选项,多个选项可以合并

参数:可以有零个,一个或者多个参数.多个参数用空格隔开

使用linux命令的小技巧

  • 查看历史命令

    • 使用上下箭头查看已经使用过的历史命令

    • 使用history命令查看已经使用过的命令

      !加序号可以执行对应的历史命令

  • 命令补全

  • 令补全是指当前输入的字符已经足够确认某个命令或者某个文件时,按tab键可以自动补全某个命令或者某个文件剩下的部分

  • ctrl加c:终止命令的执行

  • 查看命令帮助方式

  • n --help:查看命令使用

  • n man命令:表示查看指定命令的使用说明书

    man 命令名

操作说明
空格显示下一屏信息
*回车**显示下一行*
上下箭头显示上一行下一行
q退出

linux文件和目录相关命令

  • linux目录结构

    • /:根目录
    • /home:一级子目录
    • /home/python:二级子目录

  • 目录切换

    • 相对路径和绝对路径

      相对路径:是指从当前目录开始的路径

      绝对路径:从根目录/开始的路径

    • 使用cd命令切换目录

命令说明
cd 目录(相对路径或绝对路径)切换到指定目录
cd ~切换到当前用户主目录
cd ..切换到上一级目录

查看目录的文件列表信息

  • 文件信息

    • 当创建一个文件的时候,系统保存了有关该文件的全部信息:文件的名称,文件的位置,文件的类型,文件长度,文件的拥有者,文件的修改时间,文件权限等

  • ls

选项说明
-l以列表的方式查看更多文件信息
-a显示隐藏文件和隐藏目录

  • ll:查看更多文件的更多信息

  • 文件类型(ls -l查看时第一个符号)

    • d:目录
    • l:符号链接(指向另一个文件,类似于快捷方式)
    • b:块设备文件
    • c:字符设备文件
    • -:普通文件,不属于上面几种类型的文件

  • 文件权限

    • 九位字母表示法

      • r:读权限

        w:写权限

        x:执行权限

        -:无权限

        1-3位:文件属主权限

        4-6位:同组用户权限

        7-9位:其他用户权限

        例:rw-rw-r--

        创建文件的用户有读和写,同组用户有读和写,其他用户有读权限

    • 九位二进制表示法

      • 1代表有权限,0代表无权限

      • 1-3:表示创建文件的用户的读写执行

        4-6:同组用户的读写执行

        7-9:其他用户的读写执行

        例:110110100

        创建文件的用户有读和写,同组用户有读和写,其他用户有读权限

    • 三位八进制表示法

      • 4:读权限, 2:写权限, 1:执行权限

        第一位:创建文件的用户的读写执行

        第二位:同组用户的读写执行

        第三位:其他用户的读写执行

        例:754

        创建文件的用户有读写执行,同组用户有读和执行,其他用户有读权限

创建,删除及目录命令

  • 创建文件: touch 文件路径/文件名
  • 创建目录(文件夹) mkdir 目录/目录名
  • 删除文件 rm 目录/文件
  • l 删除目录 rm -rf 目录

复制,移动文件及目录

  • cp:拷贝文件或目录 cp 源文件 目录路径/目标文件名

    • 如果不指定目录文件名,使用源文件名
    • 如果目录路径下存在源文件同名文件,会覆盖目标目录中的文件

  • cp -r 源目录 目标目录

  • mv:移动文件,移动目录,重命名

    • mv 源文件 目标路径/目标文件名

      • 如果不指定目录文件名,使用源文件名

      • 如果目录路径下存在源文件同名文件,会覆盖目标目录中的文件

  • mv 原目录 目标目录

  • mv 源文件名 目标文件名(在一级目录下操作)

管道命令和重定向

  • 重定向:把原本输出到屏幕上的内容,输出到文件中(文件不存在,则创建)

    • 覆盖模式重定向:>

    • 追加模式重定向:>>

  • 管道命令:**| **将前一个程序的输出作为后一个程序的输入

查看文件内容

  • cat:查看文件内容
  • more:分屏查看文件内容,按空格切换到下一页
