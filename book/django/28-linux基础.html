@login_required
def user_edit(request):
    back_dict = {'code': 0}
    if request.method == 'POST':
        username = request.POST.get('username')
        email = request.POST.get('email')
        phone = request.POST.get('phone')
        site_name = request.POST.get('site_name')
        site_title = request.POST.get('site_title')
        site_theme = request.FILES.get('site_theme')

        if not re.match('\w+@\w+\.\w+', email):  # 校验邮箱
            back_dict['code'] = 1001
            back_dict['email'] = '邮箱格式不正确'
            return JsonResponse(back_dict)

        if not re.match('1[3-9]\d{9}', phone):  # 校验手机号
            back_dict['code'] = 1002
            back_dict['phone'] = '手机格式不正确'
            return JsonResponse(back_dict)

        # 修改userinfo表数据
        models.UserInfo.objects.filter(username=request.user.username).update(username=username,
                                                                              email=email,
                                                                              phone=phone)
        # 修改blog表数据
        blog_obj = models.Blog.objects.filter(userinfo__username=request.user.username).first()
        blog_obj.site_name = site_name
        blog_obj.site_title = site_title
        blog_obj.site_theme = site_theme  # FileField修改属性需要通过对象名.属性名=值
        blog_obj.save()
        return JsonResponse(back_dict)

@login_required
def set_avatar(request):
    blog = models.Blog.objects.filter(userinfo__username=request.user.username).first()
    if request.method == 'POST':
        avatar = request.FILES.get('file')
        request.user.avatar = avatar
        request.user.save()
        return redirect('/home/')
    return render(request, 'backend/set_avatar.html', locals())

linux

linux是一款操作系统,免费,开源,安全,高效,稳定,处理高并发非常强悍,现在有很多项目都部署到了linux服务器上运行

  • linux的版本

    • 内核版:是操作系统内部操作和控制硬件设备的核心程序.比如磁盘管理,内存管理等等
    • 发行版:一些大的公司在内核的基础上,结合各种常用的软件,形成了linux发行版
    • 常见的发行版:ubuntu,centOS,debian,redhat等等

  • linux的应用领域

    • 个人桌面领域的应用:此领域是传统的linux应用最薄弱的环节,传统的linux由于界面简单,操作复杂,应用软件少的缺点,一直被windows压制,但是近些年来,随着ubuntu等优秀桌面环境的兴起,linux在此领域的占有率逐渐提高
    • 服务器领域:linux在服务器领域的应用是最强的
    • 嵌入式领域:linux运行稳定,对网络的良好支持性,低成本,且可以根据需要进行软件裁剪,内核最小可以达到几百kb.(主要应用:机顶盒,数字电视,手机,智能家居)

  • linux和windows的比较

比较windowslinux
免费和收费收费且很贵免费或少许费用
软件和支持数量和质量的优势,不过大部分为收费软件;由微软官方提供支持开源自由软件,用户可以修改定制再发布,由于基本免费没有资金支持,部分软件的质量不太好.由全球的所有linux开发者提供支持
安全性三天两头打补丁安装系统安全更新,还是会中病毒没有绝对的安全,只是比windows更加安全
使用习惯普通用户基本都是图形界面下使用.用户上手比较简单兼具图形和命令行操作.可以只用键盘完成一切操作,上手比较困难
可定制性封闭的,系统可定制性很差开源,可定制性非常强
桌面操作系统主要使用的是windows世界上大部分软件和服务都是运行在linux之上的
