表单

是网页或应用程序中用于收集用户输入信息的一种交互元素, 这些信息将被发送到服务器，服务器端脚本或应用程序将对这些信息进行处理。

常见的用途包括但不限于以下几个方面

  • 用户注册和登录
  • 数据收集与提交
  • 搜索功能
  • 文件上传

表单属性

<form action="地址" method="get" target="_blank" name="表单名">
  <!-- 各种表单控件元素 -->
</form>

<form>元素表示文档中的一个区域，此区域包含交互控件，用于向 Web 服务器提交信息

  • action标签属性 --- 表单提交时，应该往哪个地址提交
  • method标签属性 --- 表单提交数据的方式，一般有两种 get 和 post
  • target标签属性 --- 提交数据后，后端返回的新页面在哪里打开，取值和a里面的target一样
  • name标签属性 --- 表单名

表单控件

表单控件的作用是采集用户的数据然后提交到后端，所有的控件都必须有name及value属性，并且除了单选按钮之外，所有的控件name都不能相同。

<input type="text" name="名字" />
  • type 表单控件的类型
    • text：用于接收单行文本输入。
    • radio：用于创建单选按钮，允许用户在一组互斥的选项中选择一个选项。
    • submit：用于提交表单的按钮。
    • reset：用于重置表单的按钮，将表单字段恢复到默认值。

控件通用属性

  • name: 控件名称
  • value : 设置输入框的初始值 除非type属性的值为radio或checkbox。
  • placeholder 填写提示
  • autocomplete 表单自动填充特性提示
    • off 不允许
    • on 允许
  • autofocus 自动聚焦
  • required 必须填写
  • disabled 禁止选中 禁止输入
  • checked 默认勾选

单双选择框

  • disabled: 禁止选中 禁止输入
  • checked : 默认勾选

单选框

<!-- case 3 : 单选按钮，一组选项name必须相同，value必须不同（value值随便定义），一般需要后面写文字辅助选择 -->
<input type="radio" name="sex" value="男" />男
<input type="radio" name="sex" value="女" />女
<input type="radio" name="Country" value="0" />中国
<input type="radio" name="Country" value="1" />美国
<input type="radio" name="Country" value="2" />日本
<input type="radio" name="Country" value="3" />其他

label 辅助input进行选择

<input type="radio" name="sex" value="1" id="sex1" /><label for="sex1">男</label>
<input type="radio" name="sex" value="2" id="sex2" /><label for="sex2">女</label>

<label>
  选择一个选项：<input type="text" name="myInput">
</label>
  • <label> 元素可以与关联的表单元素建立关联，以便用户点击标签时，会自动聚焦（focus）到相应的表单元素或选中相应的复选框/单选按钮
  • 无论是使用 for 和 id 属性，还是将 <input> 元素嵌套在 <label> 元素中，都可以实现与 <input> 元素的关联，使得点击标签时能够触发与之关联的表单元素的行为

多选框

<!-- case 4 : 多项选择 因为可能会有多个，所以每一项name都不能一样，value值可以一样也可以不一样 -->
<input type="checkbox" name="hobby1" value="篮球" />篮球
<input type="checkbox" name="hobby2" value="足球" />足球
<input type="checkbox" name="hobby3" value="羽毛球" />羽毛球
<input type="checkbox" name="hobby4" value="乒乓球" />乒乓球
<input type="checkbox" name="hobby5" value="网球" />网球

<!-- 点击文字,聚焦控件 -->
<label>
    <input type="checkbox" name="hobby1" value="篮球" />篮球
</label>
<label>
    <input type="checkbox" name="hobby2" value="足球" />足球
</label>

其他控件

select 和 option配合实现下拉选框

<!-- name规定给select  value规定给option并且不能相同 -->
<select>
  <option value="option1">选项1</option>
  <option value="option2" selected>选项2</option>
  <option value="option3">选项3</option>
</select>
  • value 属性定义了选项的值
  • selected 设置默认选中
  • select标签设置size属性, 规定select下拉时显示几个

下拉分组

<select>
  <optgroup label="组1">
    <option value="option1">选项1</option>
    <option value="option2">选项2</option>
  </optgroup>
  <optgroup label="组2">
    <option value="option3">选项3</option>
    <option value="option4">选项4</option>
  </optgroup>
</select>

textarea 文本域

多行文本输入框，rows标签属性定义高度，cols标签属性定义宽度；

如果需要用来传值，同样的需要制定name属性。

其他type

  • color：用于选择颜色的输入。

  • number：用于接收数值输入，可以包含整数或浮点数。

    • min最小值
    • max最大值
    • step倍数

  • date：用于接收日期输入。

  • time：用于接收时间输入。

  • range 数字滑块

  • min最小值

  • max最大值

  • file：用于上传文件。

    • multiple 选择上传多个

表格

表格（Table）是在 HTML 中用于展示结构化数据的一种元素。它由行和列组成，可以呈现出类似于电子表格的布局

简单的表格

一般我们在使用表格时，需要用到的标签并不多，table、tr、th、td；

tr定义表格的行，内容都是写在行里面的；

th定义在行里面，字体会加粗，代表一列的标题；

td定义在行里面，代表每一项；

能设置对应高度, 宽度设置不生效, 如果需要设置对应宽高 可以通过设置table进行设置对应宽高

<table>
  <tr>
    <th>姓名</th>
    <th>年龄</th>
    <th>性别</th>
  </tr>
  <tr>
    <td>John Doe</td>
    <td>25</td>
    <td>男</td>
  </tr>
  <tr>
    <td>Jane Smith</td>
    <td>30</td>
    <td>女</td>
  </tr>
</table>

边框和边距

给table加上CSS样式border-collapse:collapse;以使相邻单元格的边框合并，美化样式；

给table加上CSS样式*border-spacing: 像素值 以使每个单元格之间加上间隙

单元格合并

rowspan 属性规定单元格纵跨的行数（纵向合并）

<td rowspan="2"></td>

colspan 属性规定单元格横跨的列数（横向合并）

<td colspan="2"></td>
