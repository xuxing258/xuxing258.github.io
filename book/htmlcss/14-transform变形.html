transform

用于对元素进行转换（变换）操作，包括平移、缩放、旋转、倾斜等。通过使用 transform 属性，您可以改变元素的外观和位置，而无需影响文档流

  • 平移与relative类似

基础属性

平移 Translate

transform:translate(x,y) 通过指定 x 轴和 y 轴的偏移量，将元素在平面上移动

  • translateX( ) 默认
  • translateY( )
  • translateZ( )
.element {
  transform: translate(100px, 50px);
}

旋转 Rotate

transform:rotate(deg) 通过指定旋转角度，使元素绕其原点旋转

  • rotateZ( ) 默认是Z轴
  • rotateX( )
  • rotateY( )
.element {
  transform: rotate(45deg);
}

缩放 Scale

transform:scale(x,y) 通过指定 x 轴和 y 轴的缩放比例，增大或缩小元素的尺寸

  • skewX()
  • skewY()
  • 负数将进行倒置
.element {
  transform: scale(1.5, 2);
}

倾斜 Skew

transform:skew(x,y) 通过指定 x 轴和 y 轴的倾斜角度，使元素在水平和垂直方向上倾斜

  • skew(x , y)
  • skewX()
  • skewY()
.element {
  transform: skew(10deg, -5deg);
}

变化原点

定义了变换操作的参考点或中心点，围绕该点进行平移、缩放、旋转、倾斜等变换, 默认位置正好位于元素的水平中心和垂直中心线的交叉点, 默认值为transform-origin:50% 50%;

通过更改 transform-origin 的值，您可以控制变换操作的中心点，从而影响元素的变换效果

属性值:

  • 百分比
  • top left right bottom center
  • 像素值
  • 等其他单位

变换顺序组合

多个变换可以通过组合在一起，形成一个变换序列。通过指定变换的顺序，您可以控制它们应用于元素的顺序和效果。变换顺序是从左到右解析

.element {
  transform: rotate(45deg) scale(1.5) translate(100px, 50px);
}

在这个示例中, 首先应用了旋转变换（rotate），然后是缩放变换（scale），最后是平移变换（translate）

3D变形

通过使用 3D 变换，您可以在 X、Y 和 Z 轴上进行平移、旋转、缩放和倾斜，以创建具有立体感的效果

X轴：左右方向

Y轴：上下方向

Z轴：垂直于屏幕面的方向

（使用浏览器调试功能查看下列属性对显示效果的影响）

/* 设置近大远小 墙下看到的是2d 远处看的时3d*/
.box{
    width: 500px;
    height: 100px;
    background-color: red;
    margin: 50px auto;
    transform-origin: 0px 0px;
    transform: rotateY(0deg);
}
body{
    perspective: 1000px;   
}
  • 3D观看需要以旁观者的角度观察
    • 通过设置在父级设置perspective: 单位, 近大远小
.box {
    position: relative;
    width: 200px;
    height: 200px;
    margin: 100px auto;
    animation: move 8s linear infinite;
    transform-style: preserve-3d;
}
.box div {
    position: absolute;
    width: 100%;
    height: 100%;
    border: 1px solid #000;
    perspective: 1000px;
}

.box div:nth-child(1){
    transform: translateX(-100px) rotateY(90deg);
}
.box div:nth-child(2){
    transform: translateX(100px) rotateY(90deg);
}
.box div:nth-child(3){
    transform: translateY(-100px) rotateX(90deg);
}
.box div:nth-child(4){
    transform: translateY(100px) rotateX(90deg);
}
.box div:nth-child(5){
    transform: translateZ(100px);
}
.box div:nth-child(6){
    transform: translateZ(-100px);
}
@keyframes move {
    to{
        transform:  rotateX(360deg)  rotateY(360deg);
    }
}
  • perspective
    • 设置观察着位置, 初始是设置在父元素进行远距离观看
    • 设置3D之后可在对应元素位置进行观察, 子元素自身
  • perspective-origin
  • 设置景深视觉基点，默认在元素中央。
  • transform-style
  • 父元素设置transform-style: preserve-3d; 指示元素的子元素应位于 3D 空间中
