基础选择器

写法名称举例
*通配选择器*{ margin:0; padding:0; }
标签名元素选择器div{ width:50px; height:50px; }
.class名class选择器.nav{ width:20px; }
#id名id选择器#header{ width:50px; }
父 子 孙……后代选择器#header div p{ margin:10px; }
选择器 , 选择器组合选择器（多个选择器使用同一样式）#nav p,#tab a{ font-size: 12px; }
父 > 子子元素选择器（只会选取儿子）div>a{ color:red; }
兄 + 弟相邻兄弟选择器（紧挨着哥哥的弟弟）p + a{ color:red; }
兄 ~ 弟关联选择器（元素后续的元素）p ~ a{ color:red; }

属性选择器

允许您根据元素的属性及其属性值来选择目标元素

写法名称举例
[attr]选择具有指定属性的元素[title]{ color:red; }
[attr=val]指定属性并且属性值与给定值完全匹配的元素[target=_blank]{ color:red; }
[attr*=val]指定属性并且属性值包含给定值的元素[src*=baidu]{border:5px solid pink;}
[attr$=val]指定属性并且属性值以给定值结尾的元素[src~=jpg]{ border:5px solid pink; }
[attr^=val]指定属性并且属性值以给定值开头的元素[class^="btn"]{ color:red; }

伪类选择器

用于选择元素的特定状态或位置

写法介绍举例
:hover鼠标悬停在元素上的状态a:hover{color:pink;}
:link未被访问的链接（特指a标签）a:link{color:red};
:visited被访问过的链接（特指a标签）a:visited{color:blue;}
:active元素被激活（鼠标按下未释放或触摸屏幕按下）的状态a:active{color:green;}
:focus元素获得焦点的状态input:focus { }
:checked（单选/多选）表单被勾选状态input:checked{}
::selection文字选中状态.txt::selection{ }

结构性伪类选择器

写法介绍举例
e:first-childe元素父级所有子级中的第一个元素p:first-child {font-weight: bold;}
e:last-childe元素父级所有子级中的最后一个元素p:last-child {font-weight: bold;}
e:nth-child(n)e元素父级所有子级中的第n个元素p:nth-child(1) {font-weight: bold;}
e:nth-of-type(n)e元素父级所有子级中的第n个元素e:nth-of-type(1) {font-weight: bold;}

详情

  • e:nth-child(n)

    选中 e元素父级所有子级中的第n个元素

  • :nth-child( n+2) 从第2个开始选择后面的所有元素

  • :nth-child( -n+11) 从第一个元素开始选择到11个元素

    • nth-child(8n) 表示选取父元素下每个位置索引为 8 的倍数的子元素

  • e:nth-of-type(n)

    • 选中: e元素父级所有子级中的第n个元素 无视其他不关于e元素的元素

  • e:nth-child(even) even 偶数

    • 选中: e元素父级所有子级中的 偶数 元素

  • e:nth-child(odd) odd 奇数

    • 选中: e元素父级所有子级中的 奇数 元素

伪元素选择器

伪类选择器相当于在某种情况下添加一个虚拟类名，而伪元素选择器则是相当于创建了一个虚拟元素。

写法介绍举例
E::before相当于在E元素的最前面添加一个额外的子元素#wrap::before{content:"Hello World!"}
E::after相当于在E元素的最后面添加一个额外的子元素#wrap::after{content:"Hello World!"}

必须拥有 content 样式，上述两个伪元素才会生效。

创建的子元素是一个行内元素。

::after常用来清除浮动。

content 除了写文字之外，还可以用url指定一张图片等其他写法。
