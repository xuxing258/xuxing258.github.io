圆角

border-radius : 元素的边框创建圆角效果；

取值通常使用百分比值 或 像素值 ；

值最多可为4个：

四个值： 左上 右上 右下 左下

三个值：左上 右上左下 右下

两个值：左上右下 右上左下

一个值：四个角同一值

圆角可以分解为更细的分解属性，比如左上角：border-top-left-radius3

div {
  border-radius: 10px; /* 四个角都具有 10px 的圆角 */
}

div {
  border-top-left-radius: 5px;
  border-top-right-radius: 10px;
  border-bottom-left-radius: 15px;
  border-bottom-right-radius: 20px; /* 每个角具有不同的圆角半径 */
}

border-radius: x纵轴 x纵轴 x纵轴 x纵轴/y横轴 y横轴 y横轴 y横轴
			水平线左上   右上    右下   左下  /  垂直线左上   右上    右下   左下

阴影

通过使用 box-shadow 属性，你可以为元素创建投影效果，使其看起来浮起来或具有深度感

box-shadow: x y blur spread color;

div{
    box-shadow:10px 5px 10px 5px red outset;
    /*
        第一个值：水平偏移（必需）
        第二个值：垂直偏移（必需）
        第三个值：模糊距离（可选，默认0）   从扩散边缘开始，向外逐渐模糊的距离
        第四个值：扩散距离（可选，默认0）   规则从何处开始模糊，默认0也就是盒子边，大于0则会看到阴影明显漏出，允许负值
        第五个值：隐藏颜色（可选，默认黑色）
        第六个值：阴影扩散方向（可选，默认ouset向外扩展，可以改为inset向内扩展）
    */
}
  • blur: 模糊距离（可选，默认0), 并且是一个渐变的过程
    • 从元素盒子的边界轮廓开始，在这个px的区域内，阴影会从清晰的边缘逐渐变得模糊、变淡，呈现出一种柔和的过渡效果
  • spread: 扩散距离（可选，默认0）
    • 在前面模糊半径所营造出的已经有模糊效果的阴影基础上，让阴影在各个方向（以元素为中心向外）边框再额外整体向外扩展
    • spread设置的值从边框开始添加

透明

opacity属性规定元素的透明度，取值 0（完全透明） ~ 1（完全不透明）；

鼠标样式

使用 cursor 属性来改变鼠标指针的样式。这可以用于在不同的交互情况下，改变鼠标在特定元素上的外观

  • auto：浏览器自动选择适合当前上下文的光标样式。
  • default：默认的箭头指针。
  • pointer：手形指针，表示链接可以被点击。
  • crosshair：十字准星，表示可以进行选中操作。
  • move：移动指针，表示元素可以被拖动。
  • text：文本指针，表示可编辑文本。
  • not-allowed：禁止符号，表示禁止进行某个操作。

渐变

使用渐变（gradient）来创建元素的背景或文本颜色的渐变效果。CSS提供了两种类型的渐变：线性渐变和径向渐变

线性渐变

  • 指在两个及以上的颜色之间呈现一种平滑的过渡效果, 指定渐变的起点和终点，并定义颜色的分布方式
  • 该效果是矢量效果，不会因为页面的放大而出现模糊的情况

  • 基础写法

    /*百分比值也可以用确却的像素值代替*/
 background: linear-gradient(red, blue);
 background: linear-gradient(red 0%, blue 100%);

  • 方向控制 渐变方向默认从上往下（to bottom），我可以在写颜色值之前加上方向控制：

    background: linear-gradient(to right, red, blue);
/*
    向上 to top
    向右 to right
    向左 to left
    向右上角 to right top
    向右下角 to right bottom
    向左下角 to left bottom
    向左上角 to left top
*/

    还可以使用角度值来指定方向（默认向下相当于180deg）：

    backgorund: linear-gradient(45deg, #fb3664 0%, #4af5db 100%);

  • 指定多个颜色和它们之间的位置

    background: linear-gradient(to right, red 0%, yellow 50%, green 100%);
/*将创建一个从红色到黄色再到绿色的线性渐变背景，其中黄色将在渐变的 50% 处出现*/

径向渐变

径向渐变（radial gradient）是一种从一个中心点向周围辐射出颜色平滑过渡的渐变效果。你可以通过指定渐变的中心点、半径和颜色分布方式来创建径向渐变

  • radial-gradient(shape position, color-stop)
    • shape: 指定渐变的形状，可以是关键字或形状值。可选值包括：
      • circle：圆形渐变（默认值）
      • ellipse：椭圆形渐变
    • position: 指定渐变的中心点位置，可以是关键字或具体的位置值。可选值包括：
      • at center：渐变的中心点位于元素的中心（默认值）
      • at top, at bottom, at left, at right：渐变的中心点位于元素的顶部、底部、左侧或右侧
      • 具体位置值：使用百分比、像素值或关键字来指定中心点的位置，例如 at 50% 50% 表示渐变的中心点位于元素的水平和垂直方向上的 50% 处
    • color-stop: 指定颜色的分布和位置。你可以指定一个或多个颜色停止点，并使用百分比、像素值或关键字来定义它们的位置。例如，red 0%, blue 50%, green 100% 表示在渐变的 0% 处为红色，50% 处为蓝色，100% 处为绿色
background: radial-gradient(red, blue);
/*将创建一个从红色到蓝色的径向渐变背景。默认情况下，渐变从中心点向四周辐射*/

background: radial-gradient(circle at center, red, blue);
/* 将创建一个以元素中心为中心点的圆形径向渐变背景*/

background: radial-gradient(ellipse at center, red, blue)
/* 将创建一个以元素中心为中心点的椭圆形径向渐变背景*/

background: radial-gradient(circle at center, red 0%, blue 50%, green 100%);
/*  将创建一个以元素中心为中心点的圆形径向渐变背景，渐变从红色开始，到蓝色，最后到绿色 */

渐变色重复

重复渐变的颜色和位置

repeating-linear-greadient 和 repeating-radial-gradient 可以让设置的背景色重复。

/* 重复的线性渐变*/
repeating-linear-gradient( to top ,red 0px,red 10px ,blue 10px, blue 20px )

/* 重复的径向渐变*/
repeating-radial-gradient(circle at  center,red 0px,red 10px ,blue 10px, blue 20px )

多次渐变

同一个元素上应用多个渐变效果

background: linear-gradient(to right, red, blue)no-repeat left / 50px 50px , radial-gradient(circle at center, yellow, green);
