文本文字属性

文本文字属性（Text-related properties）是指在CSS中用于控制文本内容样式和布局的属性。这些属性可以用于改变文本的字体、颜色、大小、对齐方式等

字体样式

  • 文字颜色

    color: 颜色值;

  • 文字大小

    font-size: px / % / em / rem;

  • 文字粗细 font-weight: 值; 值可以为：

    bold 定义粗的字体

    normal 定义正常的字体大小: 400

    100-900的整百数数值;

  • 字体类型

    font-family: 字体名称 ;

    • @font-face{ font-famity:"字体名称" ; src:url("引入字体路径") }
    • div{ font-family:"字体名称" }

  • 字体斜体 font-style: 值; 值可以为：

    normal 正常

    italic 斜体，换成一种斜体字体

  • 行高

    line-height: 数值/百分比

    • normal 默认 设置合理的行间距
    • 文字垂直居中
      • 同时会撑开元素

  • 行间距指: 文字上下两行内容区之间的间距

    • 行高 = font-size + 上间隙 + 下间隙

文本样式

  • 水平对齐方式 text-align

    left 居左对齐

    center 居中对齐

    right 局右对齐

    justify 文字向两侧对齐，对最后一行无效

  • 首行缩进 text-indent

    带单位的长度值，常用px em

  • 大小写 text-transform

    none 默认值

    uppercase 全部转化大写

    lowercase 全部转化小写

    capitalize 首字母转换为大写

  • 文本线 text-decoration

    none 没有文本线

    underline 下划线

    line-through 中划线

    overline 上划线

  • 字间距 letter-spacing

    每一个字符之间的间隙

  • 词间距 word-spacing

    单词之间的间隙

  • 换行方式 white-space

    normal 默认值，正常换行

    nowrap 文本不换行

  • 长单词或长内容 word-break

    normal 默认值，使用浏览器默认的换行方式

    break-all 允许在单词内换行

    keep-all 只能在半角空格或连字符处换行。

  • 溢出显示方式 text-overflow

    • 需要设置 overflow: hidden;
    • clip 默认，裁剪文本
    • ellipsis 显示省略符号来代表被修剪的文本

  • 多行转换省略点

    • 设置多行转换成单行

      • white-space:  nowrap  不换行
overflow: hidden;   用超出隐藏限制元素展示内容
text-overflow: ellipsis

    • 设置指定几行转换成省略号(不讲)

      • display: -webkit-box;
-webkit-line-clamp: 2;
-webkit-box-orient: vertical; 设置元素排列方式;
word-break:break-all; 长单词或数字换行

  • 垂直对齐 vertical-align

    vertical-align控制元素内部行内与行内块元素的垂直对齐方式。

    允许指定长度值和百分比值。其他取值如下：

    值描述
    baseline默认。元素的基线与父元素的基线对齐
    top元素的顶端与一行中最高元素的顶边对齐
    middle元素的中垂点与父元素的基线加1/2父元素中小写字母x的高度对齐
    bottom把元素的底端与一行中最高元素的底边对齐

| text-top | 元素的顶端与父元素字体的顶端对齐 | | text-bottom | 把元素的底端与父元素字体的底端对齐 |

只针对inline、inline-block、图片 或 表格有效

<style>
    /* 子级设置元素基线对齐, 也会影响并排其他行内与行内块元素位置*/
    .box{
        font-size: 30px;
        vertical-align: middle;
    }
</style>
<div>
    <span>xs</span><span class="box">中文</span><span>X</span> <span>13212313</span>
</div>
  • 字母的x最下方是父元素的默认基线
  • x交叉中心点为中基线.

属性继承

大部分的文本文字属性都会继承，会被子元素继承自其父元素。这意味着父元素上设置的某些属性值会自动应用到其子元素上，除非子元素上明确指定了不同的值

<style>
    /*span能够继承父级p的文字颜色，所以span显示的文字颜色也是red*/
    p{color: red;}
</style>
<p><span>span文字</span></p>

行内块对齐:(基线对齐)

  • 多个行内元素或行内块并排时,默认按照基线对齐
  • 按照基线对齐方式排列一旦其中一个行内块或者文字 上下位置发生改变, 会导致这一排的所有文字和行内块都会跟着改变
