阿里图标

阿里图标是一套包含丰富图标库的矢量图标管理平台，由阿里巴巴集团提供。它为开发者和设计师提供了免费的图标资源，可用于网页设计、移动应用开发和其他设计项目中

使用场景及原理

原理：将小图标定义成字体，通过引入字体来展示这些小图标。因为此时的小图标相当于是一个文字，所以支持文字对应的所有样式。

使用步骤

  1. 访问阿里图标官网：您可以在浏览器中打开阿里图标官方网站 阿里矢量图
  2. 注册或登录：如果您是首次访问阿里图标，您需要注册一个阿里云账号，并使用该账号登录到阿里图标平台。
  3. 浏览图标库：在阿里图标平台上，您可以浏览各种图标库，包括阿里巴巴自家的图标库以及其他用户共享的图标库。您可以通过搜索关键字、浏览分类或按照热门和最新图标进行筛选，找到需要的图标。
  4. 添加图标到项目：当找到想要使用的图标时，您可以将它们添加到自己的项目中。点击图标后，选择"加入购物车"或"添加至项目"，并选择或创建一个项目。
  5. 打开下载的文件中的 demo页面，其中会详细的介绍三种引入方式、各方式的优缺点以及对应代码。选择合适的一种方式引入到自己代码中即可。

响应式布局

响应式布局（Responsive Layout）是一种设计和开发网页的方法，使网页能够在不同设备和屏幕尺寸上提供最佳的用户体验。响应式布局可以自动适应不同的屏幕大小，包括桌面电脑、平板电脑和移动设备

实现响应式的方案

  • 方案一：媒体查询

    最好用最常用的方案。媒体查询可以规定元素在页面对应的尺寸展示对应的css样式。

    /*
    媒体类型：
        screen  用于电脑屏幕，平板电脑，智能手机等。
        all  用于所有设备
        （一般选用这两个值之一即可，其他值大部分被废弃或者用不到）

    and：
        每一个值之间的链接符号

    */

    /*初始样式*/
    body{background-color: #aaa;}


    /*屏幕宽度 小于700px 时*/
    @media screen and (max-width: 700px ){
        body{ background-color: #1dba2d;}
    }

    /*屏幕宽度 大于700px 且 小于900px 时*/
    @media screen and (min-width: 700px) and (max-width: 900px ){
        body{ background-color: #3f1dba;}
    }

    /*屏幕宽度 大于900px 且 小于1200px 时*/
    @media screen and (min-width: 900px) and (max-width: 1200px ){
        body{ background-color: #ba1d54;}
    }

    /*屏幕宽度 大于1200px 时*/
    @media screen and (min-width: 1200px ){
        body{ background-color: #ba931d;}
    }

  • 方案二：百分比布局

    尽量不实用固定的宽度，用百分比代替，从而做到随窗口缩小布局变化。但是适用性并不高，做出来的页面每时每刻都在变各种布局文字等看起来不好看。一般会配合着媒体查询在某些局部的布局使用。

  • 方案三：rem

    rem是一个相对单位， 1rem = html的font-size大小。所以可以通过js在改变页面宽度时修改html的font-size，从而做到改变页面中所有rem单位对应的样式。同方案二，适用性并不高，某些布局中宽高字体等大小不太好控制。

3. HTML5新增标签

下面列举几个较为常用的标签，其他不常用的标签大家可查阅W3C了解。

结构性语义化标签

  • header 定义了文档的头部区域
  • nav 定义文档的导航部分。
  • section 定义了文档的某个区域。比如章节、头部、底部或者文档的其他区域
  • article 用来在页面中表示一套结构完整且独立的内容部分
  • aside 定义页面的侧边栏内容。
  • footer 标签定义文档或者文档的一部分区域的尾部。

视频音频标签

  • video视频标签 的标签属性

    • src

      设置路径。

    • width/height

      设置标签的大小，类似于img，但是不会使视频文件拉伸。

    • controls

      无须值，有该属性则 显示视频控制面板。

    • muted

      无须值，有该属性则 静音。

    • autoplay

      无须值，有该属性则 自动播放。chrome为了用户体验不允许自动播放，但是允许静音下自动播放。

    • loop

      无须值，有该属性则 放完之后循环播放（默认是放完了停止）。

    • poster

      设置封面海报，需要指定一个图片地址。

    • preload

      设置页面加载后是否加载视频（当设置了autoplay时，此属性会被忽略）：

      • auto - 加载整个视频
      • metadata - 加载元数据（视频时长、尺寸大小等）
      • none - 不加载视频

  • audio音频标签 的标签属性

    • 与video标签基本一毛一样……当然，audio不支持 width、height，不支持海报poster，chrome也不支持静音播放

filter属性

filter 属性是CSS中的一个功能强大的属性，它允许您通过应用不同的视觉效果来修改元素的渲染结果。通过 filter 属性，您可以调整元素的颜色、亮度、对比度、模糊度等，以创建各种视觉效果

  • blur(): 通过模糊元素来创建高斯模糊效果。可以指定模糊半径，值越大则模糊程度越高。

    .filter-example {  filter: blur(5px);}

  • brightness(): 调整元素的亮度。可以指定百分比值，0% 表示完全黑暗，100% 表示原始亮度。

    .filter-example { filter: brightness(150%); }

  • contrast(): 调整元素的对比度。可以指定百分比值，0% 表示完全灰色，100% 表示原始对比度。

    .filter-example { filter: contrast(200%);}

  • grayscale(): 将元素转换为灰度图像。可以指定百分比值，0% 表示原始颜色，100% 表示完全灰度。

    .filter-example { filter: grayscale(50%);}

  • hue-rotate(): 通过旋转元素的色相来改变颜色。可以指定角度值，单位为度。

    .filter-example { filter: hue-rotate(90deg) }

  • invert(): 反转元素的颜色，创建颜色的负片效果。可以指定百分比值，0% 表示原始颜色，100% 表示完全反转。

    .filter-example { filter: invert(75%); }

  • saturate(): 调整元素的饱和度。可以指定百分比值，0% 表示完全无色彩，100% 表示原始饱和度。

    .filter-example { filter: saturate(200%) }

  • sepia(): 将元素转换为深褐色（旧照片）效果。可以指定百分比值，0% 表示原始颜色，100% 表示完全深褐色。

    .filter-example { filter: sepia(50%) }
