网格布局

网格布局（Grid Layout）是一种用于网页布局的CSS模块，它允许开发者以网格形式将页面划分为行和列，从而更灵活地控制元素的位置和尺寸。通过网格布局，可以轻松地创建复杂的网页布局，而无需依赖传统的浮动和定位技术

Grid 布局与Flex 布局有一定的相似性，都可以指定容器内部多个项目的位置。但是，它们也存在重大区别。

Flex 布局是轴线布局，只能指定"项目"针对轴线的位置，可以看作是一维布局。Grid 布局则是将容器划分成"行"和"列"，产生单元格，然后指定"项目所在"的单元格，可以看作是二维布局。Grid 布局远比 Flex 布局强大

基础概念

使用网格布局时，需要在网格容器上设置display: grid;来启用网格布局。然后可以通过一系列的网格布局属性来定义行和列，如grid-template-rows、grid-template-columns、grid-row、grid-column等

  • 网格容器（Grid Container）：应用网格布局的父元素，可以是任何元素，如<div>或<section>。
  • 网格项（Grid Item）：网格容器中的子元素，被布局在网格中的每个单元格内。
  • 网格线（Grid Line）：定义网格行或列的分界线。
  • 网格轨道（Grid Track）：相邻网格线之间的区域，可以是行轨道（row track）或列轨道（column track）。
  • 网格单元格（Grid Cell）：由四个相邻的网格线围成的矩形区域，包含一个网格项。
<div class="grid-container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
</div>

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 10px;
}

.item {
  background-color: #ccc;
  padding: 20px;
}

设置Grid布局

  • grid和 inline-grid会将子级转换成块元素

    • 将会与怪异盒模型类似

  • 父级控制子级

    • 继承父级宽度100%,高度自动分配,

    • 会变成块或行内元素

    • 只对子级生效

容器的属性

容器里面的水平区域称为"行"（row), 直区域称为"列"（column）

  • 容器属性宽高

    • grid-template-columns: 200px 200px 200px 设置列宽

  • grid-template-rows: 100px 100px 100px; 设置行高

    • 简便写法1

      • 避免输入重复单位, 重复几次,宽度/高度

      • repeat( 次数, 宽高 )

  • 简便写法3

  • fr单位 去除不是fr单位长度(高度) 剩下的

  • 再除以 所有fr相加的数值 = 1fr

  • 剩余的空间进行 比例分配

  • 混合使用

  • 20px repeat(6, 1fr) 20px

  • 最少尺寸

  • minmax(100px, auto) auto自适应

  • 容器属性行间隙

    • grid-row-gap: 10px
    • row-gap 新
    • grid-column-gap: 20px
      • column-gap 新
    • 复合写法
      • grid-gap: 行间隙 列间隙
      • gap: 新

  • 容器属性定义区域（area）

    • 一个区域由单个或多个单元格组成, 设置对应名称进行移动

    • grid-template-areas:'a b c' 'd e f' 'g . i'

    • 中间一列为点，表示没有用到该单元格

    • 子元素属性

      • grid-area: 容器属性名称;

      • 子元素设置名称,就移动到名称位置

    • 定义整体区域

      • grid-template-areas:'a a a' 'd d d' 'c c c'

  • 容器属性对齐方式

    • justify-content 属性， 列

    • align-content 属性， 行

      • start 对齐单元格的起始边缘。

      • end 对齐单元格的结束边缘

      • center 内部居中

      • space-around

      • space-between

      • space-evenly

  • 子元素的属性

    • 单元格位置

      • grid-column-start属性：左边框所在的垂直网格线

      • grid-column-end属性：右边框所在的垂直网格线

      • grid-row-start属性：上边框所在的水平网格线

      • grid-row-end属性：下边框所在的水平网格线

      • 单元格合并

        • grid-column-start:1;

        • grid-row-start:1;

        • grid-column-end:3;

        • grid-row-end:2;

        • 写在需要合并的的元素中

      书写网格线位置时数字和区域名: 行合并需要书写四个位置,不然位置会移动 grid-column-start: 2; grid-column-end: 3; grid-row-start: 1; grid-row-end:3; ```

      
- 选择网格线或者区域名称

  - grid-column-start: d ;
  - grid-column-end: e;

-  使用`span`关键字，表示"跨越"，即左右边框（上下边框）之间跨越多少个网格 

  -   grid-column-start: span 2;

  • 容器属性排版布局

    • grid-auto-flow:
      • row 从左到右
      • column 从上到下
    • 并且尽可能紧密填满，尽量不出现空格
      • row dense 先行后列
      • column dense 先列后行
