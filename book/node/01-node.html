JavaScript

JavaScript被定义为一种浏览器的脚本语言，一直以来其运行环境都是客户端浏览器，因为JavaScript设计的初衷就是做一些浏览器与用户的交互和一些网页的特效来补充HTML和CSS的不足

服务器

服务器是计算机的一种，它比普通计算机运行更快、负载更高、价格更贵。

  • 中心化服务提供者
    • 接收客户端的请求并提供所需的服务或资源。它承担着服务提供者的角色，通过处理和响应客户端的请求来满足用户的需求
  • 提供服务和资源
    • 服务器可以提供各种服务和资源，例如网站托管、电子邮件传输、数据库管理、文件存储和共享等。它可以是一个网站所在的主机，一个存储文件的设备，或者一个托管数据库的系统

Node

  • Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。使用事件驱动、非阻塞式 I/O 的模型，使其轻量又高效
    • 阻塞/非阻塞: 表示同步或者异步执行
  • 使用JavaScript语法，所以对于前端工程师来说学习起来相对容易,当然与原生JavaScript不同的是,原生JavaScript运行在浏览器(客户端), 而Node.js是运行在服务端的JavaScript
  • Node.js不再和浏览器页面相关，所以很多内容和原生JavaScript有着差异，比如不再有任何DOM相关操作，顶层对象不再是window等等。

命令提示符 CMD是command的缩写: 允许用户运行各种命令来控制计算机操作

  • windows
    • window + R 输入cmd 打开命令行窗口
    • 输入 node -v 查看node是否安装成功
  • mac
    • 启动台 - 其他 - 终端
    • command + 空格 进入搜索 输入终端可以找到 其他的就是一样的
  • PowerShell与cmd功能一致
