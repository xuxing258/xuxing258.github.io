笔记分析

即使是微小的进步,日积月累也会产生惊人的变化

HTML与CSS

HTML超文本标记语言, 定义了网页的结构和内容, CSS层叠样式表, 它负责网页的样式和布局

JavaScript

JavaScript 是使网页从静态文档变为动态交互式应用的关键语言。​

Node

Node.js 是一个开源、跨平台的 JavaScript 运行时环境，它让 JavaScript 能够脱离浏览器，在服务器端运行

Vue

Vue是一个优雅、灵活且易于上手的前端框架，它通过数据驱动和组件化的思想，让开发者能更专注于业务逻辑而非 DOM 操作

React

React 是一个强大、高效且灵活的 UI 构建工具，它通过组件化、声明式编程和虚拟 DOM 技术，让构建复杂交互界面变得简单可控

Python

Python是一门简单易学、功能强大的高级编程语言，被誉为“万能胶水语言”，从网站开发到人工智能都能胜任。

Spider

网络爬虫是一种按照特定规则，自动抓取互联网信息的程序或脚本。​ 它就像一只在网络上爬行的"蜘蛛"，系统地访问网页并收集数据。
