核心属性ref

将持有对实际 DOM 元素的引用，在函数组件中使用该属性：ref

不能获取函数组件,函数只能获取节点

const {useState,useRef} = React;

function MyComponent() {
    const inp = useRef(12312)

    function handleClick() {
         console.log(inp.current.value);
    }
    return (
        <div>
        <button onClick={handleClick}>Click me -</button>
		<input type="text" ref={inp} />
    </div>
);

子组件节点

forwardRef 用于向子组件转发 ref

  1. 通过使用 forwardRef，我们可以将 ref 传递给子组件，并在父组件中引用子组件的实例或 DOM 元素
  2. forwardRef 函数接受一个匿名函数作为参数，该函数接收 props 和 ref 参数,并返回组件的 JSX
const Add = forwardRef((props, ref) => {
    return <p ref={ref}>12312</p>;
});

function MyComponent() {
    const inp = useRef(12312);

    function handleClick() {
        console.log(inp.current);
        //   inp.current.value = 66666;
    }
    return (
        <div>
        <button onClick={handleClick}>	Click me - --</button>
            <Add ref={inp} />
        </div>
	);
}

事件

  • React 事件的命名采用小驼峰式（camelCase），而不是纯小写。
  • 使用 JSX 语法时你需要传入一个函数作为事件处理函数，而不是一个字符串。
  • 合成事件是 React 提供的一种跨浏览器兼容的事件系统，它封装了原生 DOM 事件，并提供一致的接口和行为

高级函数

函数式编程中的概念

  • 高阶函数
    • 如果一个函数符合下面2个规范中的任何一个，那该函数就是高阶函数。
      • 若A函数，接收的参数是一个函数，那么A就可以称之为高阶函数。
      • 若A函数，调用的返回值依然是一个函数，那么A就可以称之为高阶函数。
        • 常见的高阶函数有: Promise、 setTimeout、 arr.map()等等
  • 函数的柯里化:
    • 通过函数调用继续返回函数的方式，实现多次接收参数最后统一处理的函数编码形式。

冒泡和捕获

注册捕获阶段的事件处理函数，则应为事件名添加 Capture

<div onClickCapture={blurAdd.bind(null, "213")} >
    <button onClick={add.bind(null, "213")} >绑定传递点击</button> <br />
</div>
