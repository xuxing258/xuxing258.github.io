循环

循环是一种重复执行特定代码块的结构，使程序能够有效地处理重复性任务

for 循环

for 循环是一种在指定条件为真时重复执行代码块的循环结构

for (初始化表达式; 条件表达式; 更新表达式) {
  // 循环体代码块
}
​```

- 初始化表达式：在循环开始前执行一次的表达式，用于初始化循环变量。
- 条件表达式：用于判断循环是否继续执行的条件，如果条件为真，将继续执行循环体；如果条件为假，将退出循环。
- 更新表达式：在每次循环迭代后执行的表达式，用于更新循环变量的值。

以下是一个示例，演示如何使用 `for` 循环输出数字 1 到 5：

​````javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
​```

在上面的示例中，初始化表达式 `let i = 1` 初始化了循环变量 `i` 的初始值为 `1`。条件表达式 `i <= 5` 检查 `i` 是否小于等于 `5`，如果是，则执行循环体中的代码块，并在每次迭代后通过更新表达式 `i++` 将 `i` 的值增加 1。循环将重复执行，直到 `i` 的值不再满足条件，即大于 `5`。
  • 更新表达式: 可以递增初始变量的值或者递减递乘等
  • 避免无限循环： 条件表达式永远为true
  • break 语句用于立即终止循环，并跳出循环体
    • 当遇到 break 语句时，程序将立即退出循环，不再执行循环体中剩余的代码，而是执行循环后的代码
  • continue 语句用于跳过当前迭代，并继续执行下一次迭代
    • 当遇到 continue 语句时，程序将跳过循环体中剩余的代码，并立即进行下一次迭代

while

while 循环是一种在条件为真时重复执行代码块的循环结构

while (条件表达式) {
  // 循环体代码块
}

- 条件表达式：在每次循环迭代前判断的条件，如果条件为真，将继续执行循环体；如果条件为假，将退出循环。

以下是一个示例，演示如何使用 `while` 循环输出数字 1 到 5：

​```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
​```

在上面的示例中，首先初始化变量 `i` 的值为 `1`。然后，条件表达式 `i <= 5` 检查 `i` 是否小于等于 `5`，如果是，则执行循环体中的代码块，并在每次迭代后通过 `i++` 将 `i` 的值增加 1。循环将重复执行，直到 `i` 的值不再满足条件，即大于 `5`。
  • 使用 while 循环时，需要注意确保循环条件能够在某个时刻变为假，否则可能导致无限循环
  • break 语句用于立即终止循环，并跳出循环体
  • continue 语句用于跳过当前迭代，并继续执行下一次迭代

do while

do-while 循环是一种先执行代码块，然后在条件为真时重复执行的循环结构

do {
  // 循环体代码块
} while (条件表达式);
​```

- 条件表达式：在每次循环迭代后判断的条件，如果条件为真，将继续执行循环体；如果条件为假，将退出循环。

以下是一个示例，演示如何使用 `do-while` 循环输出数字 1 到 5：

​````javascript
let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 5);
​```

在上面的示例中，首先初始化变量 `i` 的值为 `1`。然后，执行循环体中的代码块，并在每次迭代后通过 `i的值增加 1。循环将重复执行，直到条件表达式 `i <= 5` 不再满足，即 `i` 的值大于 `5`。
  • 使用 do-while 循环时，需要注意确保循环条件能够在某个时刻变为假，否则可能导致无限循环
  • break 语句用于立即终止循环，并跳出循环体
  • continue 语句用于跳过当前迭代，并继续执行下一次迭代
  • 与 while 循环的不同之处在于，do-while 循环保证循环体至少被执行一次，即使条件一开始就为假

for-in

是一种用于遍历对象属性的循环结构

for (let 变量 in 对象) {
  // 执行操作的代码块
}
  • 变量：在每次迭代中，将用于表示对象的一个属性名。
  • 对象：要遍历的对象。
let person = {
  name: 'John',
  age: 30,
  occupation: 'Engineer'
};

for (let key in person) {
  console.log(key + ': ' + person[key]);
}

for of

循环是一种用于遍历可迭代对象的循环结构, 可迭代对象(数组)

for (let 变量 of 可迭代对象) {
  // 执行操作的代码块
}
  • 变量：在每次迭代中，将用于表示可迭代对象的一个元素。
  • 可迭代对象：需要遍历的可迭代对象，例如数组、字符串、Map、Set 等。
let arr = [1, 2, 3, 4, 5];

for (let element of arr) {
  console.log(element);
}

for-in 循环和 for-of 循环在遍历数组时有以下区别：

  1. 遍历的内容：
    • for-in 循环：遍历的是数组的索引或键，包括数组自身的可枚举属性以及原型链上的属性。
    • for-of 循环：遍历的是数组的元素值。
  2. 遍历顺序：
    • for-in 循环：遍历的顺序不是固定的，可能与数组元素在内存中的顺序不一致。
    • for-of 循环：按数组元素在内存中的顺序进行迭代，保持原始顺序。
  3. 可迭代对象限制：
    • for-in 循环：可以用于遍历任何对象，不仅限于数组。
    • for-of 循环：只能用于遍历实现了迭代器接口的可迭代对象，如数组、字符串、Map、Set 等。
