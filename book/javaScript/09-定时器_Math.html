定时器

定时器（Timer）是JavaScript中用于延迟执行代码或定期执行代码的机制。它允许你在一定的时间间隔后执行特定的操作或函数

延迟定时器

setTimeout(callback,Time)：该函数用于在一定的延迟时间后执行一次指定的函数或代码块

setTimeout(function() {
  // 在延迟时间后执行的代码
}, 延迟时间);

间隔定时器

setInterval(callback,Time)：该函数用于按照指定的时间间隔重复执行指定的函数或代码块

setInterval(function() {
  // 按照时间间隔重复执行的代码
}, 时间间隔);

动画帧

用于执行动画和执行需要与浏览器重新绘制同步的操作的方法. 提供了一种优化的方式来执行动画效果，以及在浏览器重新绘制之前执行代码的能力。

requestAnimationFrame 的优势在于它会自动适应浏览器的刷新率，避免无效的绘制和过度的 CPU 使用。它还会在页面处于非激活状态时自动停止，以节省资源

function animate() {
  // 执行动画的代码
  requestAnimationFrame(animate);
}
// 启动动画
requestAnimationFrame(animate);

清除

  • 使用 clearTimeout 清除使用 setTimeout 创建的定时器
  • 使用 clearInterval 清除使用 setInterval 创建的定时器
  • 使用 cancelAnimationFrame 清除使用 requestAnimationFrame 创建的动画帧
let timeoutId = setTimeout(function() {}, 延迟时间);
clearTimeout(timeoutId); // 清除 setTimeout 定时器

let intervalId = setInterval(function() {}, 时间间隔);
clearInterval(intervalId); // 清除 setInterval 定时器

function animation() {
  requestAnimationFrame(arguments.callee);
}
let animationId = requestAnimationFrame(animation);
cancelAnimationFrame(animationId); // 清除 requestAnimationFrame 动画帧

单线程

在任意给定的时刻只能执行一个任务。单线程执行的含义是代码按照顺序依次执行，每个操作都必须等待前一个操作完成后才能执行

线程阻塞

由于 JavaScript 是单线程的，长时间运行的任务会阻塞主线程，导致页面失去响应，这被称为阻塞或长时间运行的 JavaScript 任务,

为了解决这个问题，可以使用异步编程模型，如回调函数、Promise、async/await 等，将耗时的操作交给其他线程（如 Web Worker）或通过事件循环的机制在后台执行。

需要注意的是，虽然 JavaScript 主线程是单线程执行的，但浏览器环境提供了其他线程，如网络线程、渲染线程、定时器线程等，以处理异步任务、渲染页面和执行定时器操作，以提高整体的性能和用户体验

for(var i = 0; i< 10000;i++){
		console.log(1);
}
console.log(2);
// js的是单线程的也就是说上面的代码,无论for循环多么耗时，
// 都必须先执行完,在打印2,这就会造成线程阻塞的问题

执行线程

  • 解析和加载：JavaScript引擎首先会解析代码，检查语法错误，并加载相应的脚本文件。
  • 全局上下文创建：在代码执行之前，会创建全局执行上下文，包括全局变量、函数声明等。
  • 顺序执行：从上到下逐行执行代码，包括变量声明、函数调用、表达式计算等。
  • 函数调用：当遇到函数调用时，会创建一个新的函数执行上下文，并将其推入调用栈，执行函数内部的代码。
  • 异步操作：如果代码中存在异步操作（如定时器、事件监听、网络请求等），它们将被放入事件队列等待执行，直到主线程空闲时才会执行异步操作的回调函数

Math

提供了一系列数学相关的方法和属性，用于进行数学计算和操作

  • Math.abs(x): 返回一个数的绝对值。
  • Math.ceil(x): 返回大于或等于给定数的整数。
  • Math.floor(x): 返回小于或等于给定数的整数。
  • Math.round(x): 返回给定数的四舍五入值。
  • Math.max(x1, x2, ..., xn): 返回一组数中的最大值。
  • Math.min(x1, x2, ..., xn): 返回一组数中的最小值。
  • Math.random(): 返回一个介于 0（包括）和 1（不包括）之间的随机数。
  • Math.sqrt(x): 返回给定数的平方根。
  • Math.pow(x, y): 返回 x 的 y 次幂。
  • Math.sin(x), Math.cos(x), Math.tan(x): 返回给定角度的正弦、余弦、正切值。
  • Math.PI: 表示圆周率 π 的常量
  • Math.trunc(4.1) 保留整数部分 简写 ~~4.1
  • Math.sign() 用于获取数值的符号
    • 当 x 为正数时，sign 方法返回 1；当 x 为负数时，返回 -1；当 x 为零时，返回 0。如果 x 是 NaN（非数字）或无法转换为数值，则返回 NaN

注意: 自动转换成数字类型,否则返回NaN, 接受固定参数,存在多个参数,自动忽略后续参数

var x = -5;
console.log(Math.abs(x)); // 输出: 5

var y = 4.7;
console.log(Math.ceil(y)); // 输出: 5
console.log(Math.floor(y)); // 输出: 4
console.log(Math.round(y)); // 输出: 5

console.log(Math.max(2, 8, 5)); // 输出: 8
console.log(Math.min(2, 8, 5)); // 输出: 2

console.log(Math.random()); // 输出: 0.12345（随机数）

console.log(Math.sqrt(16)); // 输出: 4

console.log(Math.pow(2, 3)); // 输出: 8

console.log(Math.sin(Math.PI / 2)); // 输出: 1
console.log(Math.cos(Math.PI)); // 输出: -1
console.log(Math.tan(0)); // 输出: 0

console.log(Math.PI); // 输出: 3.141592653589793

console.log(Math.trunc(4.9)); // 输出: 4

三角函数

Math 对象提供了一些常见的三角函数，可以用于计算三角形的相关值

  • Math.sin(x): 返回给定角度 x（以弧度为单位）的正弦值
    • 对边比斜边
  • Math.cos(x): 返回给定角度 x（以弧度为单位）的余弦值
    • 邻边比斜边
  • Math.tan(x): 返回给定角度 x（以弧度为单位）的正切值
    • 对边比邻边
  • Math.asin(x): 返回给定值 x 的反正弦值（以弧度为单位）
  • Math.acos(x): 返回给定值 x 的反余弦值（以弧度为单位）
  • Math.atan(x): 返回给定值 x 的反正切值（以弧度为单位）

示列:

let angle = 45; // 角度
let radians = angle * (Math.PI / 180); // 将角度转换为弧度
let sinValue = Math.sin(radians); // 计算正弦值
console.log(sinValue); // 输出: 0.7071067811865476

let angle = 60; // 角度
let radians = angle * (Math.PI / 180); // 将角度转换为弧度
let cosValue = Math.cos(radians); // 计算余弦值
console.log(cosValue); // 输出: 0.5

let angle = 30; // 角度
let radians = angle * (Math.PI / 180); // 将角度转换为弧度
let tanValue = Math.tan(radians); // 计算正切值
console.log(tanValue); // 输出: 0.5773502691896257

Math.sin(60*Math.PI/180 )  = 对边比斜边

- y = 150 * Math.sin(60*Math.PI/180) 
- x = Math.cos(60*Math.PI/180)*150

弧度与角度

弧度（radian）和角度（degree）都是用于测量角度的单位，但它们之间存在一种转换关系

  • 弧度（radian）：是一个角所对应的弧长与半径之比。一个完整的圆周对应的弧长为 2πr，其中 r 是圆的半径。弧度的符号为 "rad"。
  • 角度（degree）：是将一个圆等分为 360 个部分，每个部分对应一个角度单位。角度的符号为 "°"

弧度和角度之间的转换关系可以通过下述公式进行计算：

  • 弧度到角度的转换：角度 = 弧度 × (180 / π)
  • 角度到弧度的转换：弧度 = 角度 × (π / 180)
