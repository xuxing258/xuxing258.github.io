构造函数

构造函数是一个普通的函数，用于创建和初始化对象。它通过使用 new 关键字来调用，并在调用时执行以下操作：

  1. 创建一个空对象。
  2. 将新创建的对象作为 this 上下文。
  3. 执行构造函数中的代码，初始化对象的属性和方法。
  4. 返回新创建的对象。

构造函数示列

function Person(name, age) {
  this.name = name;
  this.age = age;

  this.greet = function() {
    console.log(`Hello, my name is ${this.name} and I'm ${this.age} years old.`);
  };
}

// 使用构造函数创建对象
let person1 = new Person("Alice", 25);
let person2 = new Person("Bob", 30);

// 调用对象的方法
person1.greet(); // 输出: Hello, my name is Alice and I'm 25 years old.
person2.greet(); // 输出: Hello, my name is Bob and I'm 30 years old.

构造函数this

构造函数中的 this 关键字表示正在创建的实例对象。

当使用 new 关键字调用构造函数时，构造函数内部的 this 将绑定到新创建的对象上

function Person(name, age) {
  this.name = name;
  this.age = age;

  this.greet = function() {
    console.log(`Hello, my name is ${this.name} and I'm ${this.age} years old.`);
  };
}

let person = new Person("Alice", 25);
person.greet(); // 输出: Hello, my name is Alice and I'm 25 years old.

原型

每个构造函数对象都有一个原型，它充当对象的模板，包含可以被对象实例共享的属性和方法

构造函数可以通过原型继承属性和方法。原型形成了对象之间的原型链，使得对象可以访问其原型中定义的属性和方法，以及原型链上其他级别的属性和方法

通过构造函数的 prototype 属性来定义构造函数的原型, 在构造函数的原型上添加属性和方法

function Person(name) {
    this.name = name;
}
let person1  = new Person("Alice")

Person.prototype.greet = function() {
  console.log(`Hello, my name is ${this.name}`);
};

let person1 = new Person("Alice", 25);
let person2 = new Person("Bob", 30);

person1.greet(); // 输出: Hello, my name is Alice 
person2.greet(); // 输出: Hello, my name is Bob
// 我们通过将方法 greet 添加到 Person.prototype 上，使得通过 Person 构造函数创建的所有实例对象都可以共享 greet 方法

通过将方法添加到构造函数的原型上，我们可以避免在每个实例对象中重复定义相同的方法，节省了内存空间。而且，当我们在实例对象上访问 greet 方法时，JavaScript 引擎会自动沿着原型链查找并执行原型上的方法

proto属性

实例对象才具有 __proto__ 属性，用于指向其构造函数的原型;

注意: __proto__ 已经更改为 Object.getPrototypeOf() 操作方式

  • Object.getPrototypeOf(person1) 获取原型
function Person(name, age) {
  this.name = name;
  this.age = age;
}

let person1 = new Person("Alice", 25);
let person2 = new Person("Bob", 30);

console.log(person1.__proto__ === Person.prototype); // 输出: true
console.log(person2.__proto__ === Person.prototype); // 输出: true


// 在上述示例中，我们定义了构造函数 Person，它接受 name 和 age 参数，并将这些属性绑定到新创建的实例对象上。
// 通过 new Person("Alice", 25) 和 new Person("Bob", 30)，我们创建了两个实例对象 person1 和 person2。
// 然后，我们可以通过 person1.__proto__ 和 person2.__proto__ 访问它们各自的原型对象。这两个原型对象都是来自构造函数 Person 的 prototype 属性。

原型链

构造函数原型链是 JavaScript 中实现对象继承的一种机制。

在访问对象的属性或方法时，如果对象自身没有定义该属性或方法，JavaScript 引擎会沿着原型链向上查找，直到找到对应的属性或方法或者到达原型链的顶端

原型链的本质

在 JavaScript 中，每个对象都有一个原型属性 __proto__ , 指向其构造函数的原型对象。原型对象本身也是一个对象，也有自己的原型，这样就形成了一个原型链

function Person(){	}	
let xiaoming = new Person();

// 原型对象的原型对象
console.log(xiaoming.__proto__.__proto__);

// 原型对象的原型对象的构造函数是谁
console.log(xiaoming.__proto__.__proto__.constructor); //Object

// 那我们看看还能不不能再向上查原型对象
console.log(xiaoming.__proto__.__proto__.__proto__);
// 结果为null

原型链终点

在 JavaScript 中，原型链的终点是 Object.prototype, 几乎所有的对象都是从 Object 对象通过原型继承而来的，包括自定义对象和内置对象。Object.prototype 是所有对象的顶层原型，即原型链的最终层级

let arr = new Array()
arr.__proto__.__proto__  // object

let num = new Number()
num.__proto__.__proto__  // object

原型继承

通过使用构造函数和原型链来实现对象之间的继承关系的方法。它允许一个构造函数的实例对象继承另一个构造函数的原型上的属性和方法

// 父类构造函数
function Animal(name) {
  this.name = name;
}

// 在父类的原型上定义方法
Animal.prototype.sayHello = function() {
  console.log("Hello, I'm " + this.name);
};

// 子类构造函数
function Dog(name, breed) {
  // 调用父类构造函数，继承父类的属性
  Animal.call(this, name);
  this.breed = breed;
}

// 将子类的原型设置为父类的实例，实现原型继承
Dog.prototype = Object.create(Animal.prototype);
Dog.prototype.constructor = Dog;

// 在子类的原型上定义方法
Dog.prototype.bark = function() {
  console.log("Woof!");
};

// 创建子类对象
var dog = new Dog("Buddy", "Labrador");

dog.sayHello(); // 输出: Hello, I'm Buddy
dog.bark();     // 输出: Woof!

在上述示例中，我们有一个子类构造函数 Dog，它接受 name 和 breed 参数。在子类的构造函数中，我们使用 Animal.call(this, name) 调用父类构造函数，以继承父类的属性。

接下来，我们使用 Object.create(Animal.prototype) 将子类的原型对象设置为父类的实例，以实现原型继承。这样，子类的原型就能够访问父类原型上的方法。

最后，我们在子类的原型上定义了一个 bark 方法。

通过这种方式，我们可以创建一个 Dog 的实例 dog，它继承了 Animal 的属性和方法。我们可以调用 dog.sayHello() 和 dog.bark() 来验证继承是否成功。
