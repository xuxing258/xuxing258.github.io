节点类型

在 DOM（文档对象模型）中，每个节点都有一个类型，表示节点在文档中的不同角色和功能, 可以通过节点的 nodeType 属性来判断节点类型

element.childNodes 获取元素的子节点

节点名称nodeType描述
元素节点1表示 HTML 元素，如 <div>, <p>, <a> 等
属性节点2表示元素节点的属性。 以键值对的形式存在，如 id="myElement"
文本节点3表示元素节点中的文本内容
实体引用名称节点5一种将特殊字符表示为实体的机制,例&lt; 表示小于符号 <
实体名称节点6通常用于表示特殊字符、预定义的字符串或外部实体资源
处理指令节点7它们通常以 <? 开始，以 ?> 结束，并位于文档的开头或其他位置
注释节点8表示 HTML 中的注释内容
文档节点9表示整个 HTML 文档
文档类型节点10表示文档的类型声明，如 <!DOCTYPE html>
文档片段节点11表示document.createDocumentFragment()
DTD声明节点12它通常位于文档的开头，并使用 <!DOCTYPE html PUBLIC....> 标记进行声明
  • nodeName: 表示节点的名称，对于元素节点，它是标签名；对于属性节点，它是属性名称；对于文本节点，它是 #text 等等。
  • nodeValue: 表示节点的值，对于文本节点，它是文本内容；对于属性节点，它是属性值。
  • nodeType: 表示节点的类型，是一个整数值，用常量来表示不同的节点类型，例如 Node.ELEMENT_NODE 表示元素节点，Node.TEXT_NODE 表示文本节点，
  • attributes: 表示元素节点的属性列表，以 类数组 的形式返回

节点进阶API

常用:

  • children 属性是父节点的只读属性，返回一个包含父节点的所有元素子节点的 HTMLCollection。它只包括元素节点，而不包括其他类型的节点

  • parentNode 属性是一个只读属性，表示当前节点的父节点

  • offsetParent 属性是只读属性，返回一个指向最近的已定位（positioned）祖先元素（非 static 定位）的引用。如果没有已定位的祖先元素，则 offsetParent 是根元素（<html>）

不常用:

  • firstElementChild / firstChild: firstElementChild 属性返回父节点的第一个子元素节点（不包括文本节点和注释节点），而 firstChild 属性返回父节点的第一个节点，可能是元素节点、文本节点、注释节点等。
  • lastElementChild / lastChild: lastElementChild 属性返回父节点的最后一个子元素节点（不包括文本节点和注释节点），而 lastChild 属性返回父节点的最后一个节点，可能是元素节点、文本节点、注释节点等。
  • nextElementSibling / nextSibling: nextElementSibling 属性返回下一个兄弟元素节点（不包括文本节点和注释节点），而 nextSibling 属性返回下一个兄弟节点，可能是元素节点、文本节点、注释节点等。
  • previousElementSibling / previousSibling: previousElementSibling 属性返回前一个兄弟元素节点（不包括文本节点和注释节点），而 previousSibling 属性返回前一个兄弟节点，可能是元素节点、文本节点、注释节点等

创加删换节点

创建

  • createElement(tagName) 方法用于创建一个指定标签名的元素节点
  • createTextNode(text) 方法用于创建一个包含指定文本内容的文本节点
  • createComment(text) 方法用于创建一个包含指定文本内容的注释节点
  • createDocumentFragment 方法用于创建一个空的文档碎片节点

添加

  • appendChild 方法用于将一个新的子节点添加到指定元素的子节点列表的末尾。它接受一个参数 newNode，表示要添加的新节点。返回添加的新节点
  • insertBefore(newNode, referenceNode)方法用于将一个新的子节点插入到指定元素的子节点列表中的某个已存在的子节点之前

替换

  • replaceChild(newNode, oldNode) 用于用新节点替换指定元素的子节点, 返回被替换的旧节点

删除

  • removeChild(childNode) 方法是在父节点上调用的方法，用于从父节点中移除指定的子节点
  • remove()方法是在节点自身上调用的方法，用于从 DOM 树中移除当前节点，即删除元素自身。被删除的节点将不再在文档中存在

get与query区别

  • get是动态获取 get获取的是动态集合,在插入节点,删除,替换 都会检测到 动态获取的元素会随文档改变获取

  • query是静态获取 静态集合(快照)，而静态的不会获取出来之后就和文档的改变无关了。

BOM

BOM (Browser Object Model) 是指浏览器对象模型，它提供了一组用于操作浏览器窗口、文档和其他组件的对象和方法。BOM 是 JavaScript 与浏览器之间的接口，通过 BOM，JavaScript 可以与浏览器进行交互和控制

重要事件

  • onresize 浏览器窗口或元素的大小改变时触发
  • onscroll 当页面滚动或元素的滚动条滚动时触发

open

open 函数可以用来打开新的浏览器窗口或标签页

window.open(url, target, features, replace);
​```

- `url`：指定要在新窗口或标签页中加载的 URL。
- `target`（可选）：指定打开方式，默认为 `_blank`，在新窗口或标签页中打开。
- `features`（可选）：指定窗口的特性，如大小、位置、工具栏等。
- `replace`（可选）：指定是否替换浏览器历史记录中的当前页面。

例如，下面的代码将在新标签页中打开一个名为 "example" 的网页：

​````javascript
window.open('https://example.com', 'example');
​```

close

用于关闭窗口或标签页的方法, 在浏览器环境中，可以使用 window.close() 方法关闭当前窗口或标签页。这个方法只能关闭由 JavaScript 打开的窗口

location

location提供了与当前窗口的 URL 相关的信息和操作

  • location.href：获取或设置当前页面的完整 URL。
  • location.protocol：获取或设置当前页面的协议（如 "http:" 或 "https:"）。
  • location.host：获取当前页面的主机名和端口号。
  • location.hostname：获取当前页面的主机名。
  • location.port：获取当前页面的端口号。
  • location.pathname：获取当前页面的路径部分。
  • location.search：获取当前页面的查询参数部分（即 URL 中的问号后面的部分）。
  • location.hash：获取或设置当前页面的片段标识符（即 URL 中的井号后面的部分）。
  • location.assign(url)：加载指定的 URL，跳转到新页面。
  • location.replace(url)：用指定的 URL 替换当前页面，无法通过浏览器的后退按钮返回。
  • location.reload()：重新加载当前页面

history

用于在浏览器中访问和操作浏览器的历史记录。通过 history 对象，可以实现在用户访问过的页面之间进行导航、前进和后退等操作

  • history.length：获取当前历史记录中的页面数量。
  • history.back()：导航到上一个页面，相当于用户点击浏览器的后退按钮。
  • history.forward()：导航到下一个页面，相当于用户点击浏览器的前进按钮。
  • history.go(delta)：导航到相对于当前页面的偏移量为 delta 的页面。例如，history.go(-1) 相当于 history.back()，history.go(1) 相当于 history.forward()。

尺寸获取

可视区宽高

  • window.innerWidth
  • window.innerHeight
    • 包含滚动条的宽度
  • document.documentElement.clientWidth
  • document.documentElement.clientHeight
    • 不包含滚动条

元素宽高

  • element.clientWidth element.clientHeight
  • 是用于获取元素的可见宽度和高度的属性
    • 包括内边距但不包括滚动条的宽度: 宽(高)+padding
  • element.offsetWidth element.offssetHeight
    • 是用于获取元素的完整宽度和高度的属性
    • 包括内容区域、内边距、边框的宽度: 宽(高)+padding+border。
    • 滚动条（如果存在）计算出包含自身滚动条宽或高
      • 元素滚动条是占据内容部分生成
  • element.scrollWidth element.scrollHeigh
  • 是用于获取元素内容的完整宽度和高度的属性
    • 包括内容区域的宽度以及通过滚动显示的不可见部分的宽度: 宽(高)+padding+实际宽高

元素距离

  • element.offsetLeft element.offsetTop
    • 获取当前元素左边（上边），到定位父级的左边（上边）的距离。
  • element.getBoundingClientRec()
  • 提供了元素的大小及其相对于视口的位置 {top:100,left:20,bottom:500,right:890}。
  • element.offsetParent 找到距离自身最近具有定位样式的祖先元素

滚动距离

  • element.scrollTop element.scrollLeft

    • 用于获取或设置元素(页面)滚动条的垂直和水平滚动位置的属性

      let element = document.getElementById("myElement");
let scrollTop = element.scrollTop;
let scrollLeft = element.scrollLeft;

console.log("Scroll top: " + scrollTop);
console.log("Scroll left: " + scrollLeft);

// 设置滚动位置
element.scrollTop = 100;
element.scrollLeft = 50;

  • window.pageYOffset window.pageXOffset

  • 用于获取文档在垂直和水平方向上相对于视口的滚动偏移量的属性。

  • element.clientLeft element.clienTop

  • 用于获取元素的边框宽度和边框高度

自定义窗口位置

element.scrollTo(x,y | options) 用于将元素(页面)滚动到指定的位置

  • top 指定沿 Y 轴滚动窗口或元素的像素数。
  • left 指定沿 X 轴滚动窗口或元素的像素数。
  • behavior
    • smooth 表示平滑滚动并产生过渡效果、
    • instant 表示滚动会直接跳转到目标位置，没有过渡效果。
    • auto 或缺省值表示浏览器会自动选择滚动时的过渡效果。
let element = document.getElementById("myElement");

// 将元素滚动到指定位置
element.scrollTo({
  top: 0,
  left: 100,
  behavior: "smooth"
});

// 页面滚动
document.documentElement.scrollTo({  top: 1000, left: 0, behavior: "smooth"});
window.scrollTo({  top: 1000, left: 0, behavior: "smooth"});

element.scrollBy(x,y | options) 使得元素滚动一段特定距离
