Promise

promise是用于处理异步的解决方案。表示一个异步操作的最终完成或失败，在操作完成后返回结果或抛出错误

Promise特点

Promise 必然处于以下几种状态之一：

  • 待定（pending）： 初始状态，既没有被兑现，也没有被拒绝。
  • 已兑现（fulfilled）：意味着操作成功完成。
  • 已拒绝（rejected）：意味着操作失败。
let myPromise = new Promise(function(res,rejected){})
console.log(myPromise) // 默认等待状态

let myPromise = new Promise(function(res,rejected){
	res("成功回调");
  	rejected("失败回调)
})
console.log(myPromise)  // 外部接受数据
  • 状态一旦发生变换则不可再更改
  • 回调函数的参数可将数据反应到外部。

then与catch

var promise = new Promise(function(res,rej){
    var a = 10;
    setTimeout(()=>{
        a = "这个才是我想要的数据"
        res(a)
    },5000)
})
promise.then(
    function(data){ console.log(data)}, // 成功
    function(err){console.log(err)}   // 失败
)

// 用获取成功和获取失败时如何处理, 以及维护和代码的简洁度讲解   ajax直接获取的区别
var promise = new Promise((res,rej) => {
    var xml = new XMLHttpRequest();
    xml.open("get", "https://jsonplaceholder.typicode.com/posts", true)
    xml.send()
    xml.onreadystatechange = function () {
        if (xml.readyState == 4) {
            var value = xml.responseText;
            if(value){
                // 数据获取成功处理
                res(JSON.parse(xml.responseText));
            }else{
                // 数据获取失败处理
                rej({code:0,value:"数据获取失败"})
            }
        }
    };
})
promise.then((data)=>{
    console.log(data);
}).catch((err)=>{
    console.log(err);
    // 获取失败 再次重新请求
})
  • 当promise状态转换成其他状态时会触发then()
    • 执行成功函数或失败函数
  • then方法可以被链接起来，形成Promise链，依次处理多个异步操作的成功情况
  • catch方法处理异步操作失败（Promise进入已拒绝状态）的情况

Promise(API)

resolve

Promise.resolve()是一个静态方法，用于创建一个已经被解决（已完成）的Promise对象，并返回该Promise对象

const resolvedPromise = Promise.resolve('Hello, world!');
resolvedPromise.then((result) => {
  console.log(result); // 输出：Hello, world!
});

reject

Promise.reject() 静态方法返回一个已拒绝（rejected）的 Promise 对象

const error = new Error('Something went wrong');
const rejectedPromise = Promise.reject(error);

rejectedPromise.catch((error) => {
  console.log(error.message); // 输出：Something went wrong
});

all

Promise.all()是一个静态方法，用于将多个Promise对象组合成一个新的Promise对象，该新的Promise对象在所有输入的Promise对象都解决（已完成）时解决，并返回一个包含所有解决值的数组

const promise1 = Promise.resolve('Hello');
const promise2 = 42;
const promise3 = new Promise((resolve, reject) => {
  setTimeout(resolve, 1000, 'World');
});

Promise.all([promise1, promise2, promise3])
    .then((results) => {
    console.log(results); // 输出：['Hello', 42, 'World']
})
    .catch((error) => {
    console.log(error);
});

race

Promise.race()是一个静态方法，用于将多个Promise对象组合成一个新的Promise对象，并返回一个新的Promise对象，该对象在第一个解决（已完成或已拒绝）的Promise对象解决时解决

const promise1 = new Promise((resolve, reject) => {
  setTimeout(resolve, 2000, 'Promise 1 resolved');
});

const promise2 = new Promise((resolve, reject) => {
  setTimeout(reject, 1000, new Error('Promise 2 rejected'));
});

Promise.race([promise1, promise2])
    .then((result) => {
    console.log(result); // 不会执行
})
    .catch((error) => {
    console.log(error.message); // 输出：Promise 2 rejected
});

async & await

async是JavaScript中的一个关键字，用于定义一个异步函数（async function）。

异步函数是一种特殊类型的函数，在函数体内部可以使用await关键字来暂停执行，并等待一个Promise解决（已完成）或其他值

async function fn(){  // async 表示异步,这个函数里面有异步的任务
    let result = await xxxx;  // 表示等待await后面的结果需要等待,等出来在处理
}
let s = fn()
  • async是与promise以及ajax配合使用(以promise和ajax的案例讲解)

async特点

  • await 只能放到async函数中使用
  • await 后面可以是promise对象,数字,字符串,布尔值等
  • 使用async关键字声明的函数会自动返回一个Promise对象
  • 在async函数中遇到await会等待其结果后才会接续往下执行
  • 只要await 语句后面的Promise的状态变成reject, 那么整个async函数会中断执行

验证async函数返回一个promise对象

async function fn(){

}
console.log(fn());  //Promise

// 使用
async function fn(){
	return 'welcome';
}
fn().then(res => {
    console.log(res);    // welcome
})

如果async函数中出错

async function fn(){
    throw new Error('Error 出错了')
}
fn().then(res => {
    console.log(res);
}).catch(err => {
    console.log(err);  //  Error 出错了
})

补充

使用 try { } catch (){}; 解决async函数中的报错

  • try{}用来检测里面的代码,如果出现报错,会直接被catch捕获
  • catch(err){} 参数err就是捕获的错误,这个错误有两个属性,name,message
// 抛出错误  throw
throw new Error('abc');

new ReferenceError()  	引用错误
new TypeError()           类型错误
new SyntaxError()      	 语法错误

例子:

async function fn(){
    try{
        await Promise.reject('出错了')
    }catch(e){
        let a = await Promise.resolve("成功了");
        console.log(a);
    }
}
