DOM

DOM（Document Object Model）文档对象模型, 它将文档表示为一个树状结构，其中每个节点都代表文档中的一个元素、属性、文本或其他内容

在 Web 开发中，JavaScript 通常用于与 DOM 进行交互，通过 DOM 可以动态地修改网页内容、样式和结构，以及响应用户交互

获取元素

JavaScript中要操作HTML标签，得先获取到对应的标签，我们称之为元素或者节点对象。

  • 通过ID获取
  • document.getElementById("main")
  • 通过选择器获取
    • document.querySelector("#main .nav")
    • document.querySelectorAll(#banner li)
  • 通过class名获取
  • document.getElementsByClassName("left")
  • 通过标签名获取
  • document.getElementsByTagName("p")
  • 特殊标签的获取
  • 获取html标签 document.documentElement
    • 获取head标签 document.head
  • 获取body标签 document.body
    • 获取title标签 document.title

一般情况下，推荐使用 ID获取 或者 选择器获取 ，比较方便。

元素内容

  • Element.innerHTML 获取元素里所有内容
  • Element.innerText 获取元素里文本内容
  • element.innerHTML = "新的内容"; 修改元素的 HTML 内容
  • element.textContent = "新的文本内容"; 修改元素的纯文本内容

操作元素属性

  • 获取节点.getAttribute("name")
  • 设置节点.setAttribute("属性名","属性值")
  • 设置节点.removeAttribute("属性名")

自定义属性

所谓自定义属性就是HTML标签原生不支持,但是开发人员为了开发方便自行给元素加上去的属性

  • 设置自定义 data属性 元素.dataset.属性="属性值"

  • 获取自定义属性 元素.dataset.属性

DOM事件

DOM 事件是指在网页中发生的各种交互和状态变化，比如用户的点击、鼠标移动、键盘输入等。通过 JavaScript，可以对这些事件进行监听并执行相应的操作

监听事件

  • 事件种类

    鼠标事件： onclick 左键单击 ondblclick 左键双击 onmouseover onmouseenter 鼠标移入 onmouseout onmouseleave鼠标移出 onmousedown 鼠标按下 onmousmove 鼠标移动 onmouseup 鼠标抬起 oncontextmenu 右键单击

    键盘事件：onkeydown onkeypress 键按下 onkeyup 键抬起

    系统事件： onload 页面加载完成后 onerror 加载出错后 onresize 窗口调整大小时 onscroll 滚动时

    表单事件： onfocus 获取焦点后 onblur 失去焦点后 onchange 改变内容后 onreset 重置后 onselect 选择后 onsubmit 提交后

  • 监听事件写法

    节点.事件 = 函数。

    eg：document.getElementById("main").onclick = function(){alert(1)};

    这里的函数称之为 事件函数，事件函数不会自执行，而是当事件触发时执行。

  • this在事件函数中，关键词 this 就表示触发事件的这个节点对象。

let oBox = doucment.getElementById("Box")
oBox.onclick=function(){
	console.log(1,this)
}

修改css样式

JavaScripts修改css样式最终是行内样式

  • 直接修改 style 属性

    • 通过直接修改元素的 style 属性，可以设置元素的各种样式属性，如背景颜色、字体大小等。
    • 注意样式属性名需要使用驼峰命名法
    let element = document.getElementById("myElement");
element.style.backgroundColor = "red";
element.style.fontSize = "20px";

  • 属性设置和删除

    let element = document.getElementById("myElement");
// 设置样式
element.setAttribute("style", "background-color: red; font-size: 20px;");
// 删除样式
element.removeAttribute("style");

  • 添加或移除 CSS 类

    let element = document.getElementById("myElement");
element.classList.add("newClass");
element.classList.remove("oldClass");
element.className = "newClass";
element.setAttribute("class", "newClass");

  • cssText

    • cssText 的本质就是设置HTML 元素的style 属性值。不需要驼峰式写法
    oBox.style.cssText="width : 200px; background-color:green;"

补充

  • className
    • element.classname 获取大写的class类名
    • element.classname="box" 节点修改class类名
  • id
    • element.id 获取id名
    • element.id=id名称 设置id名
  • classList
    • element.classList.add("class1", "class2"); 向元素添加一个或多个类名
    • element.classList.remove("class1", "class2"); 从元素移除一个或多个类名
    • element.classList.contains("active"); 检查元素是否包含指定的类名，返回布尔值。
    • element.classList.replace("oldClass", "newClass"); 将元素的一个类名替换为另一个类名。
    • element.classList.toggle("active"); 如果元素有指定的类名，则移除它；如果元素没有指定的类名，则添加它
