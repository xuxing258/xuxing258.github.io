正则

用于匹配、搜索和处理文本的强大工具。它使用一种特定的语法来描述文本模式，并根据这些模式执行各种操作，如查找、替换、验证等。

创建正则

  • 字面量方式 let reg = /bufan/，用于匹配字符串中的 "bufan" 字段。

  • 构造函数形式 let reg = new RegExp("bufan")，用于匹配字符串中的 "bufan" 字段。

正则匹配

正则

reg.test(str)，检测字符串str中是否包含reg规则，返回布尔值。

let pattern = /abc/;
let text = 'Hello abc World';

let isMatch = pattern.test(text);
console.log(isMatch);  // 输出: true

reg.exec(str)，返回包含匹配结果的数组，没有则返回null。如果正则有 g 标志，返回匹配的结果数组

let pattern = /abc/;
let text = 'Hello abc World';

let result = pattern.exec(text);
console.log(result);  // 输出: ["abc", index: 6, input: "Hello abc World", groups: undefined]

字符串

str.match(reg)，返回包含匹配结果的数组，字符串内检索指定的值，或找到一个或多个正则表达式的匹配

let pattern = /abc/;
let text = 'Hello abc World';

let matches = text.match(pattern);
console.log(matches);  // 输出: ["abc"]

str.replace(reg,str|function)，将匹配的内容用第二个参数的str或者函数返回值替换，返回替换得到的新字符串。

let pattern = /abc/;
let text = 'Hello abc World';

let newText = text.replace(pattern, 'XYZ');
console.log(newText);  // 输出: "Hello XYZ World"

标志

使用标志（flags）来修改匹配的行为。常用的标志包括：

  • g（全局匹配）：在整个字符串中进行全局匹配，而不仅仅是找到第一个匹配项。
  • i（忽略大小写）：在匹配时忽略字母的大小写。
  • m（多行匹配）：启用多行模式，允许^和$匹配每一行的开头和结尾。
  • new RegExp("abc", "i");
let pattern = /hello/gi;
let text = 'Hello world! Hello there! HELLO!';

let matches = text.match(pattern);
console.log(matches);
// 输出: ["Hello", "Hello", "HELLO"]

let str =`ds23 \ndq456 \ndr23`
let reg = /^d/g
let bol = str.match(reg) 

// 多行匹配
const str = `Line 1
  \nLine 2 Line
  \nLine 3`;
const regex = /^Line/mg; // 匹配每行开头的Line
const matches = str.match(regex);
console.log(matches);

转义

正则表达式由普通字符和特殊字符（元字符）组成。普通字符包括：字母（abcABC），数字（123），_等等

特殊字符：() [] {} ^ $ * ? \ | + .

普通字符我们可以直接拿来用，但是，特殊字符是一定要转义。

let pattern = /\./;    // 匹配点号
let text = 'Hello.world';

let isMatch = pattern.test(text);
console.log(isMatch);  // 输出: true

字符集

使用字符集来匹配某一个范围的字符，比如：

  • [abc]：匹配字符集中的任意一个字符，即 a、b 或 c。

  • [0-9]：匹配任意一个数字字符。

  • [a-z]：匹配任意一个小写字母。

  • [A-Z]：匹配任意一个大写字母。

  • [a-zA-Z]：匹配任意一个字母（包括大小写）。

  • [\u4e00-\u9fa5]，匹配任一汉字字符。

  • [0-9a-z]，匹配 数字或者小写字母 字符。

  • 负向类 [^] 简短理解,就是取反的意思

    let str = "我的n1Aaaasss4456aaa"
// 正则匹配三个字符 小写字母 数字 大写字母组成的字符
let reg = /[a-z][0-9][A-Z]/
let result = str.match(reg)
console.log(result);


let pattern2 = /[^0-9]/;
let text2 = 'Hello 123 World';
let matches2 = text2.match(pattern2);
console.log(matches2);
// 输出: ["H", "e", "l", "l", "o", " ", "W", "o", "r", "l", "d"]

  • \d 匹配任何数字 (阿拉伯数字)。相当于 [0-9]

  • \D 匹配任何非数字 (阿拉伯数字) 的字符。相当于[^0-9]

  • \s 匹配单个空白字符，包括空格、制表符、换页符、换行符和其他 Unicode 空格

  • \S 匹配除空格以外的单个字符 相当于 [^\f\n\r\t\v等]

  • \w 匹配基本拉丁字母中的任何字母数字字符，包括下划线

  • \W 匹配任何不是来自基本拉丁字母的单词字符。相当于 [^A-Za-z0-9_]

let str = "我的n1A aaasss445 6aaa"
let reg = /\D/g
let result = str.match(reg)
console.log(result);

界定字符

  • ^：匹配输入字符串的开头。
  • $：匹配输入字符串的结尾。
  • . 表示匹配任意字符，即使在字符集里面，也表示匹配任意字符。
// 密码: 必须大写字母开头
let str1 = "A123a9";
// 匹配大写字母开头 中间数字 小写字母 结束位置为数字
let reg1 = /^[A-Z][0-9a-zA-z][0-9a-zA-z][0-9a-zA-z][0-9a-zA-z][0-9]$/
// 大写开头  数字结束 密码为6位数组成
let result1 = str1.match(reg1)

let str2 = "A123a9";
// 匹配大写字母开头 中间数字 小写字母 结束位置为数字
let reg2 = /^[A-Z]....[0-9]$/
// 大写开头  数字结束 密码为6位数组成
let result2 = str2.match(reg2)

let str3 = "A123a";
// 匹配大写字母开头 中间数字 小写字母 结束位置为数字
let reg3 = /^[A-Z]....[^0-9]$/
// 大写开头  数字结束 密码为6位数组成
let result3 = str3.match(reg3)

量词

量词{}: 表示要匹配字符进行多次匹配(重复匹配)

let str2 = "A12215a9";
//大写开头  数字结束 密码为8位数组成  写起来很麻烦还是固定的 
let reg2 = /^[A-Z][0-9a-zA-z][0-9a-zA-z][0-9a-zA-z][0-9a-zA-z][0-9]$/
// 改成
let reg2 = /^[A-Z][0-9a-zA-Z]{6}[0-9]$/
let result2 = str2.match(reg2)
console.log(result2);
  • 量词普通写法
    • {n} 重复匹配n次字符
    • {n,m} 重复匹配n至m区间次字符
    • {n,} 最少匹配n次到无穷次字母
  • 特殊写法
    • {1,} 可通过+代替
    • {0,} 可通过*代替
    • {0,1} 可通过?代替
let str4 = "A9";
let reg4= /^[A-Z]\w?[0-9]$/
// 大写开头  数字结束 密码最少为8位数组成 最多不限制
let result4 = str4.match(reg4)
console.log(result4);

子项

用于提取对应规则中的数据

定义正则时，可以用()将部分规则包裹起来，这样在使用match或者exec做匹配的时候，能在得到的结果里面拿到该部分匹配的内容。

let str1 = "A12215a9";
// (选择正则匹配成功后,需要提取的数据)
let reg1 = /^[A-Z]([0-9a-zA-Z]{6})[0-9]$/
let result1 = str1.match(reg1)
console.log(result1[1]);

let str2 = "1a22A15a9";
// 字符() 提取匹配字符后面的字符
let reg2 = /[A-Z](\w{1,})/
let result2 = str2.match(reg2)
console.log(result2);
  • 或 |
let str2 = "1a22A15a9";
// 匹配大写跟着数字或小写字母后面跟着数字字符
let reg2 = /([A-Z]|[a-z])[0-9]/g
let result2 = str2.match(reg2)
console.log(result2);

断言

  • /Hello(?=World)/(?=自定) 匹配Hello字符后面紧跟的自定字符

    let pattern = /Hello(?=World)/;
let text = 'HelloWorld';

let isMatch = pattern.test(text);
console.log(isMatch);

  • /(?<=Jack)Sprat/ 匹配Jack字符之后紧跟的Sprat自定字符

    let pattern = /(?<=Jack)Sprat/;
let text = 'JackSprat';

let isMatch = pattern.test(text);
console.log(isMatch);

  • /Jack(?!98)/，(?!自定) 匹配Jack字符后面不是自定字符。

    let pattern = /(?!Jack)Sprat/;
let text = 'JackSprat';

let isMatch = pattern.test(text);
console.log(isMatch);

  • /(?<!20)19/，(?<!自定1)自定2 匹配自定2的 前面不是自定1字符

    let pattern = /(?<!Jack)Sprat/;
let text = 'JackSprat';

let isMatch = pattern.test(text);
console.log(isMatch);

  • (?=.*[A-Z]) 字符串中必须包含大写字母

    let str = "oiAqiolasfnldfg";
let reg = /^(?=.*[A-Z])\w{5,10}/g;
console.log(str.test(reg));

(?=.*[a-z])：必须包含小写字母

(?=.*[0-9])：同“(?=.*\d)”必须包含数字
