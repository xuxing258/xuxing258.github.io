JSON

JSON（JavaScript Object Notation）是一种常用的数据交换格式，它以文本形式表示结构化数据。JSON数据由键值对组成，使用大括号 {} 包围对象，使用方括号 [] 包围数组。以下是JSON的一些基本规则

JSON在数据交换和存储中被广泛使用，它易于阅读、编写和解析，并且在多种编程语言中都有良好的支持

  • 键值对：键值对由键和值组成，用冒号 : 分隔，键是一个字符串，值可以是字符串、数字、布尔值、对象、数组或null。示例："key": "value"。
'{
  "name": "John",
  "age": 30,
  "isStudent": true,
  "hobbies": ["reading", "playing guitar", "traveling"],
  "address": {
    "street": "123 Main St",
    "city": "New York"
  },
  "favoriteNumbers": [7, 13, 42],
  "isNull": null
}'

JSON使用

使用 JSON.stringify() 方法将JavaScript对象转换为JSON字符串。

let person = {
  "name": "John",
  "age": 30,
  "city": "New York"
};

let jsonStr = JSON.stringify(person);
console.log(jsonStr); 
//  字符串格式 {"name":"John","age":30,"city":"New York"}

使用 JSON.parse() 方法将JSON字符串解析为JavaScript对象

let jsonStr = '{"name":"John","age":30,"city":"New York"}';

let person = JSON.parse(jsonStr);
console.log(person);

对象

属性检查

  • obj.hasOwnProperty(prop)：检查对象是否具有指定的属性，不包括继承的属性。

    let a = {c:1,d:22,w:"你好"}
let s = a.hasOwnProperty("c") 
console.log(s) // true

  • prop in obj：检查对象是否具有指定的属性，包括继承的属性

    let a = {c:1,d:22,w:"你好"};
"c" in a  // true

属性删除

delete obj.property：通过 delete 关键字删除对象的属性，其中 obj 是对象，property 是要删除的属性名

let a = {c:1,d:22,w:"你好"}
delete a.c
console.log(a.c)

属性操作

  • Object.keys(obj)：返回一个包含对象自身可枚举属性的数组。

  • Object.values(obj)：返回一个包含对象自身可枚举属性值的数组。

  • Object.entries(obj)：返回一个包含对象自身可枚举属性键值对的数组。

  • Object.getOwnPropertyDescriptor(obj, prop)：返回一个描述指定属性的对象。

    • obj 是要检索属性的对象，prop 是要检索的属性名
    const obj = {
  name: 'John',
  age: 30,
};

const descriptor = Object.getOwnPropertyDescriptor(obj, 'name');
console.log(descriptor);

​```
value：属性的值。
writable：属性是否可写（true/false）。
enumerable：属性是否可枚举（true/false）。
configurable：属性是否可删除（true/false）

  • Object.defineProperty(obj, prop, descriptor)：定义或修改对象的属性。

    • obj 是要在其上定义或修改属性的对象，prop 是要定义或修改的属性名，descriptor 是一个包含属性描述符的对象
    const obj = {};
Object.defineProperty(obj, 'age', {
  value: 30,
  writable: false,
  enumerable: false,
});
console.log(obj.age); // 30
obj.age = 40; // 无效，因为属性是只读的
console.log(obj.age); // 30
