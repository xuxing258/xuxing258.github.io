算术运算符

用于执行基本的数学运算

  • +`：加法
  • -：减法
  • *：乘法
  • /：除法
  • %：取模（取余数）
  • ++：自增
  • --：自减

% 取模/取余

let a = 12 % 3  // a = 0
// 12/3 = 4 能被取模%后值是整数那么值为零

let b = 12%5   //b = 2
// 12/ 5= 2.4   能被取模%后值是有小数值那么值为取余
// 60 % 25= 10   取余=值被整除时所剩余的值

let c = 12%13 // c=12
//第一个数字比第二大时不会被取模和取余返回第一个值

赋值运算符

用于给变量赋值

  • =：简单赋值
  • +=：加法赋值
  • -=：减法赋值
  • *=：乘法赋值
  • /=：除法赋值
  • %=：取模赋值

比较运算符

用于比较两个值并返回一个布尔结果（true或false）

  • ==：相等
  • ===：严格相等（值和类型都相等）
  • !=：不相等
  • !==：严格不相等 （值和类型都不相等）
  • >：大于
  • <：小于
  • >=：大于等于
  • <=：小于等于

字符串比较

字符串比较时，是按位比较各自的ASCII编码

从字符串左边开始，一次比较每个字符，直接出现差异、或者其中一个串结束为止。
1" > "2"  false
"21" > "123"  true
"22" > "11111"  true

引用类型比较

比较的是内存地址。

let a = {};  let b = {};  
a == b  false

类型转化比较

  • == 会变成其类型改成相同类型
  • === 不会改成相同类型
  • != 会变成其类型改成相同类型
  • !== 不会改成相同类型
  • 布尔值会转成数字类型比较
  • NaN 不等于任何
  • 同时在比较大于、小于、小于等于、大于等于时，有数字参与的比较，也会将另外一个非数字转换为数字

逻辑运算符

逻辑运算符用于对布尔值进行逻辑操作，并返回布尔结果

  • &&：逻辑与
    • 如果所有操作数都为真，则返回真；否则返回假
  • ||：逻辑或
    • 如果任何一个操作数为真，则返回真；只有当所有操作数都为假时才返回假
  • !：逻辑非
    • 反转操作数的逻辑值。如果操作数为真，则返回假；如果操作数为假，则返回真

一元运算符

递增运算符 (++) 和递减运算符 (--) 用于增加或减少变量的值

  • 前置

    • 先加减自身然后返回值,添加到运算中得出结果,赋值.

    • let num = 5;
let result = ++num;
console.log(result);  // 输出: 6
console.log(num);     // 输出: 6

  • 后缀

    • 先进行运算,再加减自身的值

    • let num = 5;
let result = num++;
console.log(result);  // 输出: 5
console.log(num);     // 输出: 6

运算顺序

先算贴身的 (! 非 ++ 自增 --自减)~ 数学运算符~ 比较运算符~ 逻辑运算符 ~ 赋值

数据类型转换

使用不同的方法进行数据类型转换

数字类型

  • 使用 Number() 函数将值转换为数字类型。
  • 使用 parseInt() 或 parseFloat() 函数将字符串转换为数字类型

注意项：

Number()            打印0
Number( null )      打印O
Number( " ")        打印0
Number( undefined ) 打印NaN
Number( true)       打印1
Number( false)      打印0
Number([])          打印О
Number([10])        打印10
Number([10,2])      打印NaN
Number({})          打印NaN
Number("1a")    		打印NaN
Number("1")      		打印1

isNaN(Number("1a"))         
isNaN()  判断是数据是否是NaN格式

字符串类型

  • 使用 String() 函数将值转换为字符串类型
  • 任意数据类型 + " " 和字符拼接转换成字符串

注意：

string(1)       	//"1"
string([1])			//"1"
string([1,2])		//"1,2"
string({})      	// "[object object]"
string(true)	  	//"true"
string(false)   	//"false"
string(undefined)	//"undefined"
string(null)		// "null"
