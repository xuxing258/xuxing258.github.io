String内置API

  • length 是一个属性，用于获取字符串的长度。

  • charAt() 和 charCodeAt()：``charAt() 方法返回指定索引位置的字符，charCodeAt()` 方法返回指定索引位置字符的 Unicode 值

    const str = 'Hello';
console.log(str.charAt(0)); // 输出 "H"
console.log(str.charCodeAt(1)); // 输出 101

  • String.fromCharCode() 通过unicode编码返回对应的字符+

  • concat() 方法用于将多个字符串连接起来

  • **indexOf() 和 lastIndexOf()：**用于查找指定字符串或字符在原字符串中的索引位置

    • indexOf(searchValue, startIndex) 返回指定字符串在原字符串中索引位置，如果未找到匹配的字符串或字符，则返回 -1
    • lastIndexOf(searchValue, startIndex) 返回指定字符串在原字符串中最后一次出现的索引位置(从后往前检索)，如果未找到匹配的字符串或字符，则返回 -1
    const str = 'Hello, World!';

console.log(str.indexOf('o'));         // 输出 4
console.log(str.indexOf('World'));     // 输出 7
console.log(str.indexOf('Universe'));  // 输出 -1
console.log(str.indexOf('o', 5));      // 输出 8

console.log(str.lastIndexOf('o'));         // 输出 8
console.log(str.lastIndexOf('World'));     // 输出 7
console.log(str.lastIndexOf('Universe'));  // 输出 -1
console.log(str.lastIndexOf('o', 5));      // 输出 4

  • substring() 和 substr() 用于从原字符串中提取子字符串

    • substring(startIndex, endIndex) 方法返回位于两个指定索引之间的子字符串，
    • substr(startIndex, length) 方法返回从指定索引开始的指定长度的子字符串。
    const str = 'Hello, World!';

console.log(str.substring(7));          // 输出 "World!"
console.log(str.substring(7, 12));      // 输出 "World"

console.log(str.substr(7));         // 输出 "World!"
console.log(str.substr(7, 5));      // 输出 "World"

  • slice(startIndex, endIndex) 方法返回原字符串的指定部分，可以使用负数索引表示从字符串末尾开始计算

    const str = 'Hello, World!';

console.log(str.slice(7));           // 输出 "World!"
console.log(str.slice(7, 12));       // 输出 "World"
console.log(str.slice(7, -1));       // 输出 "World"
console.log(str.slice(-6));          // 输出 "World!"
console.log(str.slice(-12, -7));     // 输出 "Hello"

  • toUpperCase() 和 toLowerCase()：``toUpperCase() 方法将字符串转换为大写形式，toLowerCase()` 方法将字符串转换为小写形式

  • trim() 清除空格

  • replace(searchValue, replaceValue) 在字符串中替换指定的子字符串

    const str = 'Hello, World!';
console.log(str.replace('World', 'Universe'));  // 输出 "Hello, Universe!"

  • split(separator) 指定用于分割字符串的字符或字符串

数组内置API

  • length：返回数组的长度（元素的个数）

  • push(element1)：向数组末尾添加一个或多个元素，并返回数组的新长度

  • pop()：删除并返回数组的最后一个元素

  • shift()：删除并返回数组的第一个元素。

  • unshift(element1)：向数组开头添加一个或多个元素，并返回数组的新长度

    const arr = [1, 2, 3];

arr.push(4, 5);  	console.log(arr);  // 输出 [1, 2, 3, 4, 5]
arr.pop()   		console.log(arr);  // 输出 [1, 2, 3, 4]
arr.shift()   		console.log(arr);  // 输出 [2, 3, 4]
arr.unshift(11)  	console.log(arr);  // 输出 [11,2, 3, 4]

  • sort(compareFunction) 原地对数组进行排序

    • 返回排序后的数组
    • 如果省略compareFunction参数，则数组元素将被转换为字符串，并按照 Unicode 代码点顺序进行排序
    • 返回值大于0, 表示 a 应该在 b 后面
    • 返回值小于0, 表示 a 应该在 b 前面
    • 等于0, 位置不动
    const arr = [10, 5, 8, 2, 1];
let newArr = arr.sort((a, b)=>{
    console.log("a="+ a, "b="+b, a-b);
    if (a > b) {
        return 1;    // a > b 时返回 1，表示 a 应该在 b 后面
    } else {
        return -1;   // a <= b 时返回 -1，表示 a 应该在 b 前面
    }
});
console.log(newArr);

// 不需要进行记录 直接使用数值进行处理
let newArr = arr.sort((a, b)=> a - b)  
let newArr = arr.sort((a, b)=> b - a)  

sort使用: 插入排序：
原理：把待排序的数组分成已排序和未排序两部分，初始时已排序部分只有一个元素，即数组的第一个元素。然后从未排序部分取出一个元素，插入到已排序部分的合适位置，使得已排序部分仍然保持有序。
特点：对于接近有序的数组效率较高，同样不适合大规模数据的排序。

  • splice(start, deleteCount, item1, item2, ...)：从指定的索引位置开始修改数组，可以删除、替换或插入元素

    • start：必需，指定修改开始的索引位置。
    • deleteCount：可选，表示要删除的元素数量。如果设置为 0，则不删除任何元素。
    • item1, item2, ...：可选，要插入到数组的元素
    const arr = [1, 2, 3, 4, 5];
const removedElements = arr.splice(2, 2, 'a', 'b');

console.log(arr);              // 输出 [1, 2, 'a', 'b', 5]
console.log(removedElements);  // 输出 [3, 4]

  • reverse( ) 数组反向排序

  • concat(item1,item1)：将两个或多个数组合并为一个新数组。

  • slice(start, end)：返回数组的一部分，从指定的开始索引到结束索引（不包括结束索引）

  • join(separator)：将数组的所有元素连接成一个字符串。

    • 默认使用逗号 , 作为分隔符

  • indexOf()：返回指定元素在数组中第一次出现的索引。

  • lastIndexOf()：返回指定元素在数组中最后一次出现的索引。

  • Array.isArray(value) 判断接受的对象是否是数组

注意： splice reverse sort push pop shift unshift 都会修改原数组

高阶循环

  • forEach(callback(currentValue, index, array))：对数组的每个元素执行指定的操作，没有返回值

    • element 当前正在处理的元素。
    • index 可选，当前元素在数组中的索引
    • arr 可选，调用 forEach() 的数组对象
    const arr = [1, 2, 3, 4, 5];
arr.forEach((element, index) => {
  console.log(`Element at index ${index}: ${element}`);
});

  • map(callback(currentValue, index, array))：对数组的每个元素执行指定的操作，并返回由操作结果组成的新数组。

    const arr = [1, 2, 3, 4, 5];
const doubledArr = arr.map((element) => {
  return element * 2;
});
console.log(doubledArr);  // 输出 [2, 4, 6, 8, 10]

  • filter(callback(currentValue, index, array))：用于对数组的每个元素进行指定条件的筛选，返回满足条件的元素组成的新数组

    const todos = [
  { id: 1, task: 'Buy groceries', completed: false },
  { id: 2, task: 'Walk the dog', completed: true },
  { id: 3, task: 'Clean the house', completed: false },
  { id: 4, task: 'Do laundry', completed: true },
];

const incompleteTasks = todos.filter((todo) => {
  return !todo.completed;
});
console.log(incompleteTasks);

  • reduce(callback(accumulator, currentValue, index, array), initialValue)：对数组的每个元素执行指定的累积操作，返回最终的累积结果

    • callback：必需，表示对每个元素执行的回调函数。
      • accumulator：累积器，保存上一次回调函数的返回值或初始值。
      • currentValue：当前正在处理的元素。
      • index：可选，当前元素在数组中的索引。
      • array：可选，调用 reduce() 的数组对象。
    • initialValue：可选，作为累积器初始值的值。
    const arr = [1, 2, 3, 4, 5];
const sum = arr.reduce((accumulator, element) => {
  return accumulator + element;
}, 0);
console.log(sum);  // 输出 15

  • some(callback(currentValue, index, array))用于检测数组中是否至少有一个元素满足指定条件，返回一个布尔值

    const users = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 },
  { name: 'Charlie', age: 20 },
];
const hasTeenager = users.some((user) => {
  return user.age >= 13 && user.age <= 19;
});
console.log(hasTeenager);  // 输出 true

  • every(callback(currentValue, index, array)) 测试一个数组内的所有元素是否都能通过指定函数的测试。它返回一个布尔值

    const users = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 },
  { name: 'Charlie', age: 20 },
];
const allAdults = users.every((user) => {
  return user.age >= 18;
});
console.log(allAdults);  // 输出 true
