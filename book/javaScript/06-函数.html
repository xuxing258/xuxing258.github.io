函数

是一种可执行的 JavaScript 代码块，它接收输入（参数），执行特定任务，并返回结果。函数可重复使用，可以在程序中多次调用

声明函数

  • 函数声明

    • 使用 function 关键字来声明一个函数，并给函数命名。函数声明会被提升，即可以在函数声明之前调用

    • // 函数的定义
function functionName(parameter1, parameter2) {
  // 函数体（执行的代码块）
  // 可以包含一系列操作和逻辑
  // 可以使用参数和局部变量
  // 可以有返回语句
}

// 函数的调用
functionName(argument1, argument2);

    • functionName 是函数的名称，可以根据需要自定义。

    • parameter1、parameter2 是函数的参数，用于接收传递给函数的值。

    • functionName(argument1, argument2) 是函数的调用，通过传递参数的方式触发函数执行

  • 函数表达式

    • 将函数赋值给一个变量，以创建一个匿名函数或具名函数。函数表达式不会被提升，只能在赋值后调用

    • let functionName = function(parameter1, parameter2) {
  // 函数体
};

  • 箭头函数

    • ES6 引入的一种更简洁的函数声明方式，使用箭头（=>）来定义函数。箭头函数没有自己的 this 值，继承自外部作用域

    • let functionName = (parameter1, parameter2) => {
  // 函数体
};

参数

实参：函数执行时，小括号里面传入的数据；

形参：函数定义时，小括号里面写的变量名；

arguments : 是一个特殊的 JavaScript 对象，表示当前函数调用时传递的所有参数列表。它可以在函数内部使用，以便访问和操作传递给函数的参数

返回值

函数可以通过使用 return 语句来返回一个值或表达式作为函数的结果。

返回值可以是任何 JavaScript 数据类型，包括基本类型（如数字、字符串、布尔值）和复杂类型（如对象、数组）

function sum(a, b) {
  return a + b;
}

var result = sum(5, 3);
console.log(result); // 输出 8
  • 默认返回undefined
  • return后立即结束函数

IIFE

自执行函数，也被称为立即调用的函数表达式（IIFE），是一种在定义后立即执行的函数

自执行函数的使用场景包括：

  • 创建一个独立的作用域，避免变量污染全局命名空间。
  • 将代码块封装在函数内部，以便进行模块化开发。
  • 避免函数内部的变量被外部访问。
(function(接受数据){  return 123 })(可传递数据)    // 返回值外部变量接受
+function(){}
-function(){}
~function(){}
!function(){}

函数简写

// 箭头函数里面只有return语句
let show = () => 1;

// 大括号被解释为代码块，所以如果箭头函数直接返回一个对象，必须在对象外面加上括号
let getTempItem = id => ({ id: id, name: "Temp" });

// 箭头函数中没有arguments
let show = () => {  console.log(arguments) }
show(1,2,3,4,5); // 报错

// 箭头函数不能到构造函数
