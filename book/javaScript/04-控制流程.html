控制流程

控制流程是指根据特定条件或规则来控制程序中语句的执行顺序。

在JavaScript中，常见的控制流程语句包括条件语句和循环语句

条件语句

if 语句：用于根据条件执行不同的代码块。

if (condition) {
  // 在condition条件为真时执行的代码块
} else {
  // 在condition条件为假时执行的代码块（可选）
}

使用嵌套的 if 语句来处理更复杂的条件逻辑

let num = 5;
if (num > 0) {
  console.log("正数");
} else if (num < 0) {
  console.log("负数");
} else {
  console.log("零");
}

switch

在不同的情况下执行不同的代码块

switch (expression) {
  case value1:
    // 在 expression 等于 value1 时执行的代码块
    break;
  case value2:
    // 在 expression 等于 value2 时执行的代码块
    break;
  default:
    // 如果 expression 不等于任何一个值时执行的代码块（可选）
}

let fruit = "apple";
switch (fruit) {
  case "apple":
    console.log("苹果");
    break;
  case "banana":
    console.log("香蕉");
    break;
  default:
    console.log("其他水果");
}
  • expression 是一个表达式，它的值将与每个 case 的值进行全等于比较。
  • case 是一个匹配表达式值的标签，如果 expression 的值与某个 case 的值全相等，则执行相应代码块。
  • break 语句用于终止 switch 语句的执行，在执行完匹配的 case 代码块后跳出 switch 结构。
  • default 语句是可选的，当 expression 的值与任何 case 的值都不匹配时，将执行 default 代码块

三目运算

根据条件的真假选择不同的值或执行不同的操作

condition ? value1 : value2
// 如果condition条件为真，则返回 value1；如果条件为假，则返回 value2
  • condition 是一个表达式，用于判断条件的真假。
  • value1 是在条件为真时返回的值或执行的操作。
  • value2 是在条件为假时返回的值或执行的操作。
let num = 5;
let message = num > 0 ? "正数" : "非正数";
console.log(message);
