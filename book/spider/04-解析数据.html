jsonpath

jsonpath 是一种用于从 JSON 文档中提取数据的查询语言，类似于 XPath 对 XML 的作用。它提供了一种简洁的方式来定位和提取 JSON 数据中的特定部分

jsonpath 不能直接解构json数据,需要进行转化成字典; 还能处理 Python 中的多种原生数据结构

  • 字典（dict）：最常用的 JSON 结构
  • 列表（list）：数组形式的数据
  • 混合嵌套结构：字典和列表的任意组合
  • JSON 字符串：需先转换为 Python 对象

JsonPath语法使用

Json结构清晰，可读性高，复杂度低，非常容易匹配，下表中对应了XPath的用法。

JSONPath描述
$根节点
@现行节点
.or[]取子节点
..就是不管位置，选择所有符合条件的条件
*匹配所有元素节点
[]迭代器标示（可以在里边做简单的迭代操作，如数组下标，根据内容选值等）
[,]支持迭代器中做多选。
?()支持过滤操作.

示例： 我们以拉勾网城市JSON文件 http://www.lagou.com/lbs/getAllCitySearchLabels.json 为例，获取所有城市。

import requests
import jsonpath

url = 'http://www.lagou.com/lbs/getAllCitySearchLabels.json'
response = equests.get(url)
# 获取的是文本字典时,需要进行类型转化,转化成json格式
data = response.text 
data = json.loads(data)

# 获取的json数据
data = response.json()

child = jsonpath(data, '$.content') # 获取子级数据

bels = jsonpath(data, '$..allCitySearchLabels')  #获取后代对应数据
bels = jsonpath(data, '$..allCitySearchLabels[content]') # [] 对应子级
bels = jsonpath(data, '$..allCitySearchLabels[A,B]') # 支持迭代器中做多选

# 过滤操作?()  @表示过滤之前的位置
bels = jsonpath(data, '$..allCitySearchLabels[A][?(@.id > 650)]')

import requests
from bs4 import BeautifulSoup
import re
import time

def spider(url):
    global index

    headers = {
        'user-agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/138.0.0.0 Safari/537.36',
    }
    resp = requests.request("GET", url, headers=headers)
    xml = BeautifulSoup(resp.text, "lxml")

    html = xml.find_all("div", {"class": "zu-info"})
    price = xml.find_all("div", {"class": "zu-side"})

    parser(html, price)

    if index < 2:
        # 获取下一页面的数据 但是不要太快了
        anext = xml.find("a", {"class": "aNxt"})
        href = anext["href"]
        index = index + 1
        spider(href)
        time.sleep(2.5)
    else:
        print("结束")

def parser(html, price):

        with open("12.txt", "a", encoding="utf-8") as f:

    pass

XPath

XPath 是一种用于在 XML 和 HTML 文档中定位节点的查询语言。相比 BeautifulSoup 的查找方法，XPath 提供了更强大和灵活的定位能力

先将HTML文档转为XML文档，再用XPath查找HTML节点或元素

pip install lxml

常用符

表达式说明
/从根节点选取（绝对路径）
//从匹配选择的当前节点选择文档中的节点，而不考虑它们的位置（相对路径）
.选取当前节点
..选取当前节点的父节点
@选取属性

通过以上方式进行获取: 返回一个列表

  • xpath节点.text 获取文本内容
  • xpath('/html/body/div/h1/text()') 通过/text()获取文本内容

通配符与或

XPath 通配符可用来选取未知的 XML 元素。

通配符描述
*匹配任何元素节点。
@*匹配任何属性节点。
//html/body/div/*  # 选择div元素的所有子元素节点

//html/body/div/@*  # 选择div元素的所有属性

/html/body//*[@class] #获取后代具有class类名的元素

#选择多个
tree.xpath('//p[@class="intro"] | //div[@class="content"]')

注意: 如果标签类名存在多个值时, 需要全部写上

import requests
from lxml import etree
import re
import time

def spider(url):
    global index
    headers = {}
    resp = requests.request("GET", url, headers=headers)
    if resp.status_code == 200:
        xml = etree.HTML(resp.text)
        elements = xml.xpath('//div[@class="bot-content"]')
        page_next = xml.xpath('//*[@id="__layout"]/div/div[3]/div[1]/div[4]/div/div[2]/div/div[3]/a[2]/@href')

        for ele in elements:
            href = ele.xpath('./@data-href')[0]
            price = ele.xpath('./div[@class="price"]//text()')[0]  # 获取价格
            title = ele.xpath('./div[@class="name-pic-box"]//text()')[0]  # 获取价格
            spider_msg(href, price, title)

        index += index + 1
        if index < 8:
            spider(page_next)

    else:
        print("请求失败:", resp.status_code)
    pass


def spider_msg(href, price, title):
    time.sleep(1.5)
    headers = {}
    resp = requests.request("GET", href, headers=headers)
    if resp.status_code == 200:
        xml = etree.HTML(resp.text)
        ele = xml.xpath('//div[@class="price-box font12"]/div//text()')
        result = "".join(ele).strip()
        result = re.sub(r'\s+', "-", result)
        with open("./ce.txt", "a", encoding="utf-8") as f:
            f.write(f'店铺:{title}\n价格:{price}\n时间:{result}\n网址:{href}\n\n')

if __name__ == "__main__":
    url = "https://www.zbj.com/fw/?es=9011"
    index = 1
    spider(url)
