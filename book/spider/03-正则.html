正则

正则表达式通过 re 内置模块实现

常用正则表达式符号

  • \w 匹配字母数字及下划线
  • \W 匹配非字母数字下划线 [^0-9a-zA-Z]
  • \s 匹配任意空白字符, 等价于 [\t\n\r\f]
  • \S 匹配任意非空字符
  • \d 匹配任意数字 [0-9]
  • \D 匹配任意非数字
  • \n 匹配一个换行符
  • \t 匹配一个水平制表符: 通常用于在文本中插入固定间隔的空白
  • [] 匹配集合, 示例[a-z]
  • . 匹配除换行符（\n）以外的任意字符

示例: r 用于简化正则表达式的编写。它的作用是避免 Python 字符串的转义规则与正则表达式的转义规则冲突

import re
str = "phone: 123-4567"
exg =  r"23"
newStr = re.search(exg, str)
print(newStr.group())  # 获取匹配的数据 23


# 不加r 会报错,程序不确定你是否需要进行转义
newStr = re.search('file\^txt', 'file^txt')  # 输出: ['file^txt']
print(newStr.group())

方法

Search匹配

用于查找整个字符串的任何位置, 是一次匹配, 只要匹配成功一个结果就返回，而不是查找所有匹配的结果，它的一般使用形式如下：

re.search(pattern, string[, flags=0])

  • pattern：匹配的正则表达式
  • string： 要匹配的字符串
  • flags：标志位，用于控制正则表达式的匹配方式，如：是否区分大小写，多行匹配等
    • re.search(r'a.b', '"a\nb', re.S) 设置.可以匹配\n
    • re.S . 匹配换行符
    • re.M 多行模式
    • re.I 忽略大小写
  • 匹配成功 → 返回一个 Match对象（包含匹配信息）。 匹配失败 → 返回 None。

match

用于查找字符串的头部,它是一次匹配，只要找到了一个匹配的结果就返回，而不是查找所有匹配的结果

re.match(pattern, string[, flags=0])

  • pattern：匹配的正则表达式
  • string： 要匹配的字符串
  • flags：标志位，用于控制正则表达式的匹配方式，如：是否区分大小写，多行匹配等
  • 匹配成功 → 返回一个 Match对象（包含匹配信息）。 匹配失败 → 返回 None。
# 只检查字符串的开头，即使中间或结尾有匹配内容也会失败      
newStr = re.match('file', 'file^txt')  # 输出: ['file^txt']
print(newStr.group())

print(newStr) # 返回的是一个匹配对象
print(newStr.group())  # .group()获取匹配内容
print(newStr.span())  # 查看匹配长度

# 只检查字符串的开头，即使中间或结尾有匹配内容也会失败      
newStr = re.match('txt', 'file^txt')  # 输出: ['file^txt']
print(newStr.group())

findall

用于查找整个字符串, 是一个高效提取所有匹配项的函数，它会扫描整个字符串并返回所有非重叠匹配的结果

re.findall(pattern, string[, flags=0])

  • pattern：匹配的正则表达式
  • string： 要匹配的字符串
  • flags：标志位，用于控制正则表达式的匹配方式，如：是否区分大小写，多行匹配等
  • 匹配成功 → 返回一个列表 。 匹配失败 → 返回 []。
import re
text = "苹果 12元, 香蕉 5元, 橙子 8元"
prices = re.findall(r'\d+元', text)
print(prices)  # 输出: ['12元', '5元', '8元']  匹配多个

sub

用于替换字符串中所有匹配正则表达式的部分的函数

re.sub(pattern,replace,Sting)

  • pattern：要匹配的正则表达式（字符串或预编译对象）。
  • repl：替换的内容（可以是字符串或函数）。
  • string：被搜索的目标字符串。
  • count（可选）：最多替换次数（默认 0表示全部替换）。
  • flags（可选）：控制匹配行为的标志（如 re.I忽略大小写）
content = 'Extra stings Hello 1234567 World_This is 123 a Regex Demo Extra stings'
content = re.sub(r'\d+', '666', content)
print(content)  # 匹配到的数字全部会被替换


content = 'Extra stings Hello 1234567 World_This is 123 a Regex Demo Extra stings'
content = re.sub(r'\d+', '666', content, count=1)
print(content)  # 智能替换一次

# 在匹配替换后的内容处进行添加   \1 表示保留原字符串并追加， 空格后面为要追加的内容  r确保\1不会被转义
content = 'Extra stings Hello 1234567 World_This is a Regex Demo Extra stings'
content = re.sub('(\d+)',r'\1 3333', content)
print(content)

元字符

^ 匹配字符串的开头或取反

import re
str =re.match(r'^abc', 'abcdef')  # 匹配成功（字符串以 'abc' 开头）
str =re.match(r'^abc', '123abc')  # 匹配失败（字符串开头不是 'abc'）


re.findall(r'[^0-9]', 'a1b2')  # 匹配所有非数字字符，结果 ['a', 'b']
re.search(r'[^aeiou]', 'apple') # 匹配第一个非元音字母 'p'

$ 匹配字符串的末尾

import re
re.search(r'abc$', '123abc')    # 匹配成功（字符串以 'abc' 结尾）
re.search(r'abc$', 'abc123')    # 匹配失败（字符串结尾不是 'abc'）

管道符 |

import re
re.findall(r'cat|dog', 'I have a cat and a dog.')  # 匹配 ['cat', 'dog']

量词 {}

# 控制前面的字符匹配重复的次数
{m}：​​精确匹配 m次​​。
{m,}：​​至少匹配 m次​​（无上限）。
{m,n}：​​匹配 m到 n次​​（贪婪模式，尽可能多匹配）。

特殊写法
- {1,}  最少匹配一次,最多无限制  		 可通过+代替
- {0,}  有就匹配,不存在也不会报错  		可通过*代替
- {0,1} 最少0次,最多1次  				可通过?代替

分组或捕获子串

match = re.search(r'(\d{3})-(\d{4})', 'Phone: 123-4567')
print(match.group(1))  # 输出 '123'（第一个分组）
print(match.group(2))  # 输出 '4567'（第二个分组）

贪婪匹配

.* 会尽可能多地匹配任意字符（除换行符） 会匹配文档中的最后一个

import re
# 匹配尽可能多的字符   
content = 'Hello 1234567 World_This is a Regex Demo'

result = re.match(r'^He.*(\d+)\s.*Demo$', content)  
print(result)
print(result.group(1)) # 7

# 匹配整个字符串中的最后一个 
content = 'Hello 1234567 World_This 8 is a Regex Demo'
result = re.match(r'^He.*(\d+)\s.*Demo$', content)  
print(result.group(1)) //  8

非贪婪匹配

.*?非贪婪匹配任意字符，遇到后续能匹配的内容就停止

import re
# 匹配尽可能少的字符
content = 'Hello 1234567 World_This is a Regex Demo'
result = re.match(r'^He.*?(\d+).*Demo$', content)

print(result)
print(result.group(1)) # 1234567

转义

\表示转义，加在特殊字符的前面，表示它不是正则里的匹配符号，而是普通文本

import re
# 匹配 "file.txt" 中的点号（.）
re.match(r'file\.txt', 'file.txt')

import re
import requests

def spider():
    headers = {
        "user-agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/138.0.0.0 Safari/537.36"
    }
    resp = requests.get(url, headers=headers)
    if resp.status_code == 200:

        re_data_about = re.findall(r'<a class="fleft" href="(.*?)">(.*?)</a>', resp.text, re.S)
        re_data_msg = re.findall(r"""<span class="buy-info">.*?<a href=".*?">(.*?)</a>.*?</span>""", resp.text, re.S)

        for about, msg in zip(re_data_about, re_data_msg):
            about_url1 = about[0]
            about_url2 = about[1]
            msg_about = re.sub(r"\n",'',msg).strip()
            with open("./1.txt","a",encoding="utf-8") as f:
                f.write(f"{about_url2}--地址:{about_url1}--著名{msg_about}\n")
    else:
        print("请求失败")
    pass


if __name__ == "__main__":
    url = "https://book.douban.com/latest?subcat=%E5%85%A8%E9%83%A8"
    spider()
