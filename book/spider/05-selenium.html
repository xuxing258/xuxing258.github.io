selenium介绍

Selenium 是一个强大的浏览器自动化工具, 广泛用于网页测试和爬取动态加载内容, (主要是用于自动化测试)

自动化是指使用技术手段模拟人工，执行重复性任务,

安装 pip install selenium

官网: https://www.selenium.dev/zh-cn/documentation/webdriver/browsers/chrome/

selenium使用

  • 控制多项浏览器的启动和操作
    • webdriver.Chrome() webdriver.PhantomJS() webdriver.Firefox() webdriver.Safari()
  • 属性和方法
    • driver.page_source 获取网页源代码
    • driver.close() 关闭页面
    • driver.quit() 退出浏览器
    • driver.title 获取网页的title标签信息
# 引入webdriver驱动,启动浏览器
from selenium import webdriver 
# 启动 Chrome 浏览器 
driver = webdriver.Chrome()
# 打开网页  打开后浏览器会自动关闭
driver.get("https://www.baidu.com")  
#--------------------------------------
from selenium import webdriver 
# 引入浏览器配置,禁止自动关闭
from selenium.webdriver.chrome.options import Options
options = Options()
# 保持浏览器的打开状态
options.add_experimental_option("detach", True)
# 隐藏自动化提示栏，并禁用部分自动化检测机制
options.add_experimental_option('excludeSwitches', ['enable-automation'])
# 传入配置信息
driver = webdriver.Chrome(options=options)

# 获取网页的源码
print(driver.page_source)
print(driver.title)

获取元素

百度首页演示: https://www.baidu.com/index.htm

  • 浏览器对象. find_element(By.type, "value")
    • type: 选择对应方式定位元素
    • value: 是对应的定位值
  • 元素节点.get_attribute("outerHTML") 获取完整 HTML
  • 元素节点.text 获取元素文本内容
from selenium import webdriver 
# 导入By 根据查找网页元素
from selenium.webdriver.common.by import By
driver = webdriver.Chrome()
driver.get('https://www.baidu.com')

# 通过ID定位
element = driver.find_element(By.ID, "id_value")
# 通过 get_attribute("outerHTML")获取完整 HTML
html_content = element.get_attribute("outerHTML")

# 通过class名定位
element = driver.find_element(By.CLASS_NAME, "class_name")
# 通过HTML标签名定位
element = driver.find_element(By.TAG_NAME, "input")

# 通过链接文本定位（完全匹配）
<a href="/about">About Us</a>
element = driver.find_element(By.LINK_TEXT, "About Us")
# 通过链接文本定位（部分匹配）
<a href="/about">About Us</a>
element = driver.find_element(By.PARTIAL_LINK_TEXT, "About")

# 通过CSS选择器定位
element = driver.find_element(By.CSS_SELECTOR, "div.login > input")
# 通过XPath定位
element = driver.find_element(By.XPATH, "//input[@name='email']")

# 通过name属性定位
<input name="email" type="text">
driver.find_element(By.NAME, "email")

等待

WebDriverWait通常用于等待网页某个元素加载完毕后进行后续操作，避免出现异常

显式等待Explicit Wait: 用于在指定时间内等待某个条件成立后再继续执行后续操作, 超过时间未获取则报错

from selenium import webdriver
from selenium.webdriver.common.by import By 
# 等待页面加载完毕，寻找某些元素
from selenium.webdriver.support.wait import WebDriverWait 
from selenium.webdriver.support import expected_conditions as EC

browser = webdriver.Chrome()
browser.get('https://www.taobao.com/')

# 设置最大等待时间为10秒
wait = WebDriverWait(browser,  timeout=10)
# 等待元素加载成功,最长等待10秒, 超过时间未获取则报错
input = wait.until(EC.presence_of_element_located((By.ID,'J_SiteFooter')))

等待条件

from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from selenium.webdriver.common.by import By

wait = WebDriverWait(driver, timeout=10)

# 1. 等待元素可见
visible_element = wait.until(EC.visibility_of_element_located((By.XPATH, "//div")))

# 2. 仅检查元素存在（可能隐藏）
hidden_element = wait.until(EC.presence_of_element_located((By.CLASS_NAME, "item")))

# 3. 等待按钮可点击
login_button = wait.until(EC.element_to_be_clickable((By.LINK_TEXT, "登录")))

# 4. 验证文本内容
success_msg = wait.until(EC.text_to_be_present_in_element((By.ID, "status"), "成功"))

# 5. 等待加载动画消失
wait.until(EC.invisibility_of_element_located((By.ID, "loading")))
等待条件作用描述典型应用场景
EC.visibility_of_element_located((By.XPATH, "//div"))等待元素在DOM中存在且可见（宽度/高度>0，非隐藏）需交互的按钮、输入框等
EC.presence_of_element_located((By.CLASS_NAME, "item"))仅检查元素在DOM中存在（可能不可见或隐藏）统计元素数量或检查数据加载
EC.element_to_be_clickable((By.LINK_TEXT, "登录"))等待元素可见且可点击（Enabled状态）提交按钮、超链接等
EC.text_to_be_present_in_element((By.ID, "status"), "成功")等待元素文本内容包含指定字符串（精确或子串匹配）验证操作结果（如提示消息）
EC.invisibility_of_element_located((By.ID, "loading"))等待元素不可见或从DOM中移除（常用于等待加载动画消失）页面跳转或异步加载完成

无头模式

有效降低被网站检测为自动化的风险，适用于爬虫和自动化测试场景

参数作用典型应用场景
--start-maximized启动时最大化窗口避免元素因窗口太小导致不可见
--headless=new无头模式（不显示GUI界面）服务器环境或不需要可视化时节省资源
--user-data-dir=/path/to/profile加载指定用户数据目录（保存Cookies/扩展/历史记录）复用登录状态或自定义浏览器环境
--incognito隐身模式（不读取历史记录，新建临时会话）确保每次启动都是干净环境
--disable-gpu禁用GPU加速（旧版无头模式可能需要）解决无头模式下的兼容性问题
--window-size=1200,800设置初始窗口大小（宽×高）需要固定窗口尺寸时使用
--disable-extensions禁用所有扩展避免扩展干扰自动化操作
--no-sandbox禁用沙盒模式（Linux服务器可能需要）解决权限不足导致的崩溃问题
from selenium.webdriver.chrome.options import Options
options = Options()
options.add_argument("--start-maximized")       # 最大化窗口
options.add_argument("--headless=new")          # 无头模式  不显示浏览器

# 存储用户数据目录 (保存cookies、缓存等)  使其访问时, 能保持登录状态
options.add_argument("user-data-dir=D:/授课/1-15肥虫班/Day8/jd/data") # 必须为绝对路径，且目录需已存在

driver = webdriver.Chrome(options)
driver.get('https://www.baidu.com')

def openJDMsg(options, url):
